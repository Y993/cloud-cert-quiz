AZ-900（Azure 基礎）の難易度・勉強法と演習問題150問【無料】

合格ライン 70% ／ 制限時間 45分 ／ 収録 150問（すべて無料・登録不要）

▸ いますぐ演習をはじめる

試験概要

AZ-900はMicrosoftが提供するAzureの入門認定試験で、クラウドの基本概念からAzureの主要サービス・料金モデル・サポート体制まで幅広くカバーする。エンジニアでなくても受験できるよう設計されており、営業・マーケティング・管理職など非技術職の受験者も多い。

試験範囲はクラウドの概念（IaaS/PaaS/SaaSの違い、共有責任モデルなど）、Azureのコアサービス（コンピューティング・ネットワーク・ストレージ・データベース）、セキュリティ・プライバシー・コンプライアンス、そしてAzureの料金と管理ツールの4領域に分かれている。Fundamentals試験のため前提知識は問われないが、クラウドの用語を初めて聞く人は多少の準備が必要だ。

この試験に合格すると「Microsoft Certified: Azure Fundamentals」バッジが取得でき、有効期限は設定されていない。上位のAzure認定（AZ-104 AdministratorやAZ-204 Developerなど）に進む前の足がかりとして使う人が多い。

難易度

難易度は低め。AWS Cloud Practitioner（CLF-C02）と並んで、主要クラウドベンダーの入門試験の中では取り組みやすい部類に入る。CLF-C02との違いを挙げると、AZ-900は単純な選択問題だけでなく、ドラッグ＆ドロップで項目を分類するタイプや穴埋め形式など、出題形式が多様な点が特徴だ。慣れないと焦ることもあるので、事前に形式を確認しておく価値はある。

出題数は40〜60問で、試験時間は45分（席に座っている総時間は約65分）。合格スコアは700/1000。CLF-C02が65問・90分なのと比べると、AZ-900はやや短時間での判断が求められる。ただし問題の多くは概念理解を問うものなので、深い技術知識がなくても対応できる。

つまずきやすいのは、AzureのサービスをAWSのサービスと混同するケースだ。「Azureにおける仮想マシンはEC2に相当するが名前はAzure Virtual Machines」といった対応関係を混乱しないよう整理しておくと安心できる。

勉強法

まずMicrosoft Learn上の公式ラーニングパス「Microsoft Azure の基礎」を一通り進めるのが最短ルートだ。無料で日本語対応しており、各モジュールには理解度確認の演習問題も付いている。全体で10〜15時間程度あれば一周できる。

Microsoft Learnに加えて手を動かす練習も取り入れると理解が深まる。AzureはMicrosoftアカウントで無料トライアルに申し込めば30日間・200ドル分のクレジットが使えるため、仮想マシンの作成やストレージアカウントの操作を実際に試すことができる。テキストを読むだけより記憶への定着が早い。

仕上げとして練習問題を繰り返す。どの問題で間違えたかを記録しておき、苦手なサービス領域に絞ってMicrosoft Learnのページに戻るのが効率的だ。このページの下にある演習問題150問もその用途に使える。

勉強期間は、IT未経験なら2〜3週間、クラウドの仕事経験がある人なら1週間前後が目安になる。Azure Virtual Training Dayという無料オンラインセミナー（2日間）を受講すると受験料無料バウチャーがもらえることがある。費用を抑えたい場合は試してみる価値がある。

よくある質問

試験は何問・何分ですか？

出題数は40〜60問、試験時間は45分です。受付や説明を含む着席時間は約65分になります。

受験料はいくらですか？

日本では12,980円（税込）です。学生向けには割引価格（9,680円・税込）が用意されています。価格は改定されることがあるので、申し込み前にMicrosoftの認定資格ページか試験申込ページで最新の金額を確認してください。なお、Microsoft Azure Virtual Training Dayを受講すると無料バウチャーがもらえる場合があります。

学生割引はありますか？

はい、あります。Microsoft Learnのプロフィールで学生ステータスを設定・認証すると、多くの国・地域で学割価格が適用されます。日本も対象です（インド・中国は対象外）。

不合格になったら何日後に再受験できますか？

1回目の不合格後は24時間待てば再受験できます。2回目以降の不合格は14日間の待機が必要です。同一試験を12か月以内に受けられるのは5回までです。

合格スコアはいくつですか？有効期限はありますか？

合格スコアは1000点満点中700点です。Fundamentals認定に有効期限はなく、一度合格すれば失効しません。

日本語で受験できますか？

できます。Pearson VUEでの申し込み時に言語として日本語を選択できます。テストセンター受験・オンライン受験ともに対応しています。
