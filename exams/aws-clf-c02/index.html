試験概要

AWS Certified Cloud Practitioner（CLF-C02）は、AWS認定資格の中でもっとも入門に位置する「Foundational」レベルの試験だ。AWSの特定サービスを深く使いこなせるかではなく、クラウドの概念・AWSの主要サービス・セキュリティの基本・料金体系を、業務で話せる程度に理解しているかが問われる。

出題数は65問（うち採点対象は50問、残り15問は今後の出題検討用の未採点問題でどれかは試験中に分からない）。試験時間は90分、合格スコアは1000点満点中700点。受験料は100 USD。試験はピアソンVUEかPSI経由でテストセンター受験またはオンライン受験を選べる。

出題はドメイン4区分に分かれている。クラウドの概念（24%）、セキュリティとコンプライアンス（30%）、クラウドテクノロジーとサービス（34%）、請求・料金・サポート（12%）。比重が一番大きいのはサービス・テクノロジー領域で、EC2・S3・RDS・Lambdaなど130超のサービスが出題範囲に含まれる。

難易度

AWS認定の中では最も易しい部類に入る。IT経験がなくても1〜2か月まじめに勉強すれば合格できる水準で、合格率はおおむね6割前後と言われている。難しいというより「範囲が広い」試験だ。サービス名と用途の対応を130個近く頭に入れる必要があり、そこが唯一の山場になる。

AWS SAA（ソリューションアーキテクト – アソシエイト）と比べると、アーキテクチャの設計判断や細かい設定値は問われない。「このユースケースに適したサービスはどれか」という問いが中心で、選択肢も4択が基本。読解力と消去法が普通に使えれば、技術的な深さより広さの勝負になる。

つまずきやすいのはセキュリティドメインで、IAM・責任共有モデル・コンプライアンスプログラム（SOC2やHIPAAなど）の区別を曖昧にしたまま受けると点を落とす。もう一か所、請求とサポートプランの違い（Basicは無料・Businessは月額最低100 USD〜など）も問われやすく、名称を混同しがちなポイントだ。

勉強法

学習期間の目安は、IT未経験なら4〜6週間、エンジニア経験があれば2〜3週間が現実的なところ。最初の1週間はAWSの公式デジタルトレーニング「AWS Cloud Practitioner Essentials」（無料）を通しで見て、クラウドの概念と主要サービスの全体像をつかむ。動画は英語だが字幕があり、日本語版コンテンツも一部提供されている。

概念が一通り入ったら、問題演習に切り替える。Udemyの模擬試験（Stephane Maarek版か Tutorials Dojo版が定番）を1セット解いて、正答率と苦手ドメインを把握する。この段階で60%以下なら教材に戻り、70%台が安定してきたら次のセットへ進む。問題を解くこと自体がインプットになるので、間違えた問題の解説を読む時間を惜しまない。

サービス名と用途の紐づけは、全部を覚えようとせず「何に使うか1行で言えるか」を基準にする。S3はオブジェクトストレージ、RDSはマネージドRDB、LambdaはサーバーレスFaaS——この粒度で130個を押さえれば十分で、細かいAPIや料金単価まで暗記する必要はない。一覧を紙にまとめて眺めるより、問題の文脈で出てきたものを積み上げていく方が定着しやすい。

試験前日は新しいことに手を出さない。苦手ドメインの模擬問題を1セット解いて感触を確認し、責任共有モデルとIAMの基本（ルートアカウント・ユーザー・ロール・ポリシーの関係）だけ最終確認しておくと安心できる。試験当日は問題を最後まで読む習慣をつけておく。選択肢の微妙な差（「可用性」か「耐久性」か、「自動スケーリング」か「自動フェイルオーバー」か）が問われることが多い。

よくある質問

試験は何問・何分ですか？

65問・90分です。ただし65問のうち採点されるのは50問で、残り15問は今後の出題検討用の未採点問題です。どの問題が未採点かは試験中に分かりません。

受験料はいくらですか？

100 USDです。日本のテストセンターで受ける場合も同額で、支払い時の為替レートで円換算されます。クーポンや割引を持っている場合は受験登録画面で適用できます。

IT未経験・クラウド未経験でもいきなり受けられますか？

受験に前提資格や実務経験の要件はありません。未経験から合格した事例は多く、AWSも「AWSの知識がない状態でも6か月程度の学習で合格水準に達することを想定した試験」と位置づけています。

不合格だったら再受験はいつから可能ですか？

不合格から14日後に再受験できます。回数制限はなく、毎回受験料が必要です。

合格点は何点ですか？正答率に直すと何割くらいですか？

1000点満点中700点が合格ラインです。スコアは換算スコアなので単純な正答率とは一致しませんが、おおよそ7割前後の正答率を継続的に出せていれば合格ゾーンに入ります。

CLF-C02とCLF-C01の違いは何ですか？

2023年9月にCLF-C01からCLF-C02に改訂されました。出題対象サービス数が約64個から130個超に拡大されたのが主な変更点です。C01向けの古い教材をそのまま使うと範囲が足りないので、C02対応と明記された教材を使ってください。
