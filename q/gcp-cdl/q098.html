CDLセキュリティと運用MEDIUM単一選択

ある企業が、Google Cloudの料金と支出を管理するための機能を活用したいと考えています。月次の支出が予算上限に近づいた場合に、担当者にメール通知を送るために使用するGoogle Cloudの機能はどれですか？

  1. A. Cloud Billing の予算アラート
  2. B. Cloud Monitoring のアラートポリシー
  3. C. Cloud Scheduler のジョブトリガー
  4. D. Security Command Center の設定ミス検出
解答と解説を見る

正解: A

Cloud Billingの予算アラートは、プロジェクトまたは請求先アカウントの実際の支出または予測支出が設定した予算金額の特定割合（例：50%、90%、100%）に達した際にメールや Pub/Sub通知でアラートを送信する機能。これにより予期しないコスト増加を早期に検知できる。Cloud Monitoringのアラートはパフォーマンスメトリクス（CPU、レイテンシ等）の監視用で課金金額の監視には使用しない。Security Command Centerはセキュリティリスクの検出。Cloud Schedulerは時間ベースのジョブ実行トリガー。
