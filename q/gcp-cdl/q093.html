CDLセキュリティと運用EASY単一選択

ある企業が、Google CloudのCompute EngineとCloud StorageのパフォーマンスメトリクスとカスタムアラートをGUI上で管理したいと考えています。最も適切なGoogle Cloudのサービスはどれですか？

  1. A. Cloud Debugger
  2. B. Cloud Trace
  3. C. Cloud Monitoring
  4. D. Cloud Logging
解答と解説を見る

正解: C

Cloud Monitoringは、Google Cloudリソース（VM、GKE、Cloud SQL等）のパフォーマンスメトリクス（CPU使用率、メモリ、レイテンシ、エラー率等）を収集・可視化し、ダッシュボード作成とアラートポリシーの設定ができるサービス。閾値を超えた場合にメール・PagerDuty・Slackへの通知も設定可能。Cloud LoggingはログデータのI/O収集・検索・分析。Cloud TraceはアプリケーションのHTTPリクエストのレイテンシプロファイリング。Cloud Debuggerは本番コードのライブデバッグ（非推奨化）。
