CDLデータイノベーションとAIEASY単一選択

ある旅行会社が、多言語対応の顧客サポートを効率化するため、問い合わせメールを自動的に日本語に翻訳したいと考えています。最も適切なGoogle Cloudのサービスはどれですか？

  1. A. Cloud Speech-to-Text
  2. B. Cloud Translation API
  3. C. Dialogflow CX
  4. D. Cloud Natural Language API
解答と解説を見る

正解: B

Cloud Translation APIは100以上の言語間のテキスト翻訳に特化したサービスであり、APIコール1つで外国語メールを日本語に自動翻訳できる。既成APIのため機械学習の専門知識不要で即座に活用可能。Cloud Natural Language APIはテキスト分析（感情、エンティティ等）が主用途で翻訳機能は提供しない。Cloud Speech-to-Textは音声をテキストに変換するサービス。Dialogflow CXはチャットボット・会話フロー設計ツールで翻訳専用サービスではない（内部でTranslation APIを使うことは可能）。
