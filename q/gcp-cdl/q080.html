ある企業が、オンプレミスとGoogle Cloudを接続するためにCloud VPNを採用しました。Cloud VPNの主な特徴として最も正確なものはどれですか？

  1. A. Cloud VPNの設定は完全に自動で、ユーザーが構成する必要はない
  2. B. Cloud VPNを使うとオンプレミスとGoogle Cloudのデータは平文で転送される
  3. C. Cloud VPNは専用の物理回線を使いインターネットを経由しない通信を実現する
  4. D. Cloud VPNはIPSecプロトコルを使用してオンプレミスとGoogle Cloud VPC間に暗号化されたトンネルをインターネット経由で構築する
解答と解説を見る

正解: D

Cloud VPNはIPSecプロトコルを使ってオンプレミスのVPNゲートウェイとGoogle Cloud VPCのCloud VPN Gatewayの間に暗号化されたトンネルをインターネット経由で構築するサービス。専用線（Cloud Interconnect）より安価で、帯域幅要件が低い場合や一時的な接続に適している。専用物理回線を使わずインターネット経由がCloud VPNの特徴（専用線はCloud Interconnect）。データはIPSecで暗号化されて転送される（平文ではない）。オンプレミス側VPNゲートウェイの設定はユーザーが行う必要がある。
