CDLインフラストラクチャとアプリケーションのモダナイゼーションEASY単一選択

ある企業が、Google Cloudの「Compute Engine」を使ってWebサーバーを運用しています。トラフィックのピーク時に自動的にVMを追加し、オフピーク時には削除することで、コストを最適化したいと考えています。この機能として最も適切なものはどれですか？

  1. A. Cloud Armor
  2. B. マネージドインスタンスグループ（MIG）のオートスケーリング
  3. C. Cloud Interconnect
  4. D. Cloud CDN
解答と解説を見る

正解: B

マネージドインスタンスグループ（MIG）のオートスケーリング機能を使うと、CPU使用率、リクエスト数、カスタムメトリクス等に基づいてVMインスタンスを自動的に追加・削除できる。ピーク時には必要なVM数まで自動拡張し、オフピーク時には最小数まで縮小することでコストを最適化できる。Cloud CDNはコンテンツキャッシュサービス。Cloud ArmorはDDoS/WAFセキュリティ。Cloud InterconnectはオンプレミスとGoogle Cloud間の専用接続サービス。
