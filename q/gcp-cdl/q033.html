CDLデータイノベーションとAIMEDIUM単一選択

ある小売企業が、Webサイトの商品レコメンデーション機能を改善するためにMLを活用したいと考えています。各ユーザーの過去の購買履歴・閲覧履歴に基づいてパーソナライズされた商品提案をリアルタイムで行うために、最も適切なGoogle Cloudのサービスはどれですか？

  1. A. Cloud Translation API
  2. B. Vertex AI Recommendations AI
  3. C. Looker Studio
  4. D. Cloud Natural Language API
解答と解説を見る

正解: B

Vertex AI Recommendations AI（旧Recommendations AI）は、ユーザーの行動履歴（閲覧・購買等）に基づいてパーソナライズされた商品レコメンデーションをリアルタイムで生成するためのマネージドMLサービス。Googleのショッピング・YouTube等と同じ推薦アルゴリズム技術を活用できる。Cloud Translation APIはテキスト翻訳。Cloud Natural Language APIはテキスト分析（感情・エンティティ等）。Looker StudioはBIダッシュボードツールで、リアルタイムレコメンデーション機能はない。

📚 関連サービスの解説: Vertex AI
