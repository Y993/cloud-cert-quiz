CDLインフラストラクチャとアプリケーションのモダナイゼーションMEDIUM単一選択

ある企業が、Google Cloudに移行した後もオンプレミスとクラウドを専用線で接続し、安定した高帯域幅通信を実現したいと考えています。インターネット経由の接続よりも安定した低レイテンシ接続を提供するGoogle Cloudのサービスはどれですか？

  1. A. Cloud VPN（暗号化VPNトンネルによるインターネット経由接続）
  2. B. Cloud CDN（コンテンツ配信ネットワーク）
  3. C. Cloud Armor（Webアプリケーションファイアウォール）
  4. D. Cloud Interconnect（オンプレミスとGoogle Cloud間の専用物理接続）
解答と解説を見る

正解: D

Cloud InterconnectはオンプレミスデータセンターとGoogle Cloudを専用の物理回線（10Gbps/100Gbps）で直接接続するサービスであり、インターネット経由のトラフィックが流れないため安定した低レイテンシと高帯域幅を実現できる。大量データ転送やミッションクリティカルなハイブリッド構成に最適。Cloud VPNはインターネット経由のIPSec暗号化トンネルであり、インターネット品質に依存する。Cloud CDNはWebコンテンツのエッジキャッシュ配信。Cloud ArmorはDDoS/WAF保護。
