CDLデータイノベーションとAIMEDIUM単一選択

ある企業のデータチームが、BigQueryに蓄積された売上データを使ってビジネスユーザーが自分でダッシュボードを作成できる環境を整えたいと考えています。最も適切なGoogle Cloudのサービスはどれですか？

  1. A. Cloud Composer
  2. B. Dataflow
  3. C. Cloud Dataproc
  4. D. Looker Studio（旧Data Studio）
解答と解説を見る

正解: D

Looker Studio（旧Google Data Studio）はビジネスインテリジェンス（BI）ツールであり、BigQueryを含む各種データソースに接続して、コーディング不要でインタラクティブなダッシュボードやレポートを作成できる。ビジネスユーザーでも容易に活用できる。Cloud DataprocはHadoop/SparkのマネージドサービスでETL処理向け。DataflowはApache Beamベースのストリーム/バッチデータ処理サービス。Cloud ComposerはApache AirflowベースのワークフローオーケストレーションサービスでBIツールではない。
