CDLインフラストラクチャとアプリケーションのモダナイゼーションMEDIUM単一選択

ある小売企業が、世界中のユーザーに対して静的なWebコンテンツ（画像、CSS、JavaScript等）を高速に配信したいと考えています。最もコスト効率よく世界規模の低レイテンシ配信を実現するためのGoogle Cloudサービスはどれですか？

  1. A. すべてのリージョンにCompute Engineインスタンスを個別にデプロイしてコンテンツを複製する
  2. B. Cloud CDN（コンテンツ配信ネットワーク）をCloud Load BalancingとCloud Storageと組み合わせて使用する
  3. C. Cloud Storage単体でコンテンツを公開する
  4. D. Cloud VPNでオンプレミスのCDNサーバーとGoogle Cloudを接続する
解答と解説を見る

正解: B

Cloud CDNはGoogleのグローバルエッジネットワーク（世界140以上のPoPポイント）を活用し、静的コンテンツをユーザーに最も近いエッジにキャッシュして低レイテンシ配信を実現する。Cloud Load Balancing + Cloud StorageとCloud CDNを組み合わせることで、高速かつコスト効率の高い静的コンテンツ配信基盤を構築できる。Cloud Storage単体ではエッジキャッシュが効かず遠距離ユーザーへのレイテンシが高くなる。各リージョンへの個別VMデプロイは管理コストが高い。Cloud VPNはオンプレミスとの専用通信でCDNとは別の用途。
