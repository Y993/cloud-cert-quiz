SC-900Microsoft コンプライアンス ソリューションの機能EASY複数選択

Microsoft Purview コンプライアンスポータルから直接アクセスできるコンプライアンス機能を 2 つ選択してください。

  1. A. Microsoft Defender for Endpoint のエンドポイント検出と応答（EDR）ダッシュボード
  2. B. データ損失防止（DLP）ポリシーの作成と管理
  3. C. Microsoft Entra ID のユーザーとグループの管理
  4. D. Azure Security Center のセキュアスコアとセキュリティ推奨事項
  5. E. コンプライアンスマネージャー（規制コンプライアンスのスコアリングと改善アクション管理）
解答と解説を見る

正解: B, E

Microsoft Purview コンプライアンスポータルはコンプライアンス専用のポータルであり、コンプライアンスマネージャー（選択肢E）と DLP ポリシー管理（選択肢B）はその代表的な機能として直接アクセスできる。他にも情報保護（秘密度ラベル）・データライフサイクル管理・インサイダーリスク管理・eDiscovery・監査・コミュニケーションコンプライアンスなどが含まれる。選択肢Aの Microsoft Defender for Endpoint の EDR ダッシュボードは Microsoft 365 Defender ポータル（security.microsoft.com）で管理するものであり、コンプライアンスポータルとは別のポータルである。選択肢Cの Entra ID のユーザー管理は Microsoft Entra 管理センター（entra.microsoft.com）が担う。選択肢Dの Azure Security Center のセキュアスコアは Azure Portal 配下の Microsoft Defender for Cloud で確認するものであり、コンプライアンスポータルの機能ではない。
