SC-900Microsoft コンプライアンス ソリューションの機能MEDIUM単一選択

ある企業のコンプライアンスチームが、過去 180 日以内に発生した SharePoint ドキュメントの閲覧・ダウンロードイベントを調査する必要がある。Microsoft Purview 監査 (Standard) でこの要件を満たせるかを確認したい。Standard と Premium の違いとして正しい説明はどれか。

  1. A. Standard は Exchange のログのみを記録し、SharePoint のログは Premium のみで取得できる
  2. B. Standard と Premium の保持期間は同じで、Premium は検索速度が向上するだけである
  3. C. Standard は監査ログを 180 日間保持し、Premium は最大 1 年間または 10 年間（アドオンにより）保持する
  4. D. Standard は管理者のみ検索でき、Premium はエンドユーザーも自身のログを検索できる
解答と解説を見る

正解: C

Microsoft Purview 監査（Standard）の監査ログ保持期間は 180 日間である。監査（Premium）では保持期間が最大 1 年間（Microsoft 365 E5 ライセンス）に延長され、さらに 10 年間保持のアドオンライセンスも利用可能である。また Premium では高帯域幅の API アクセスや重要イベント（メールアクセスや個別メール送信など）の追加ログも提供される。選択肢Aは誤りで、Standard でも SharePoint・Teams などのログを記録できる。選択肢Bは誤りで、保持期間は Standard と Premium で明確に異なる（180日 vs 1年）。選択肢Dは誤りで、監査ログ検索は権限を付与された管理者・コンプライアンス担当者が行うものであり、エンドユーザーへの開放有無は Standard/Premium の差異ではない。
