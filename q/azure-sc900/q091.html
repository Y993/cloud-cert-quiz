SC-900Microsoft セキュリティ ソリューションの機能EASY単一選択

ある企業の IT 管理者が、Microsoft 365 メール環境でフィッシング攻撃やマルウェア添付ファイルから従業員を保護する最小コストかつ専用のソリューションを検討しています。この要件に最も適した Microsoft Defender 製品はどれか。

  1. A. Microsoft Defender for Office 365
  2. B. Microsoft Defender for Cloud Apps
  3. C. Microsoft Defender for Identity
  4. D. Microsoft Defender for Endpoint
解答と解説を見る

正解: A

Microsoft Defender for Office 365 は Exchange Online、SharePoint、Teams などの Microsoft 365 サービスを標的とするフィッシング・マルウェア・スパムから保護するために特化した製品です。Safe Attachments・Safe Links・フィッシング対策ポリシーを提供します。Dの Defender for Endpoint はデバイス（エンドポイント）の侵害検出・応答（EDR）が主目的であり、メールフィルタリングは行いません。Cの Defender for Identity はオンプレミス Active Directory への攻撃検出が目的です。Bの Defender for Cloud Apps はクラウドアプリの利用監視（CASB）であり、メール特有の脅威対策には不向きです。
