SC-900Microsoft コンプライアンス ソリューションの機能MEDIUM単一選択

ある法律事務所が、特定の訴訟案件に関連するメールや文書を収集・レビュー・エクスポートするための電子証拠開示（eDiscovery）プロセスを構築したいと考えている。関係者が多く、データ量も大規模であるため、カストディアン管理・高度な機械学習レビュー支援・弁護士依頼人特権の識別が必要とされる。この要件に最も適した Microsoft Purview の eDiscovery ソリューションはどれか。

  1. A. Microsoft Purview eDiscovery (Standard)
  2. B. コンテンツ検索（スタンドアローン）
  3. C. Microsoft Purview 監査（Standard）
  4. D. Microsoft Purview eDiscovery (Premium)
解答と解説を見る

正解: D

Microsoft Purview eDiscovery (Premium) は、カストディアン（データ保管者）管理、機械学習による重複除去・テーマ分析・関連性スコアリング、弁護士依頼人特権の自動識別、高度なレビューセットなどの機能を提供し、大規模・複雑な法的調査に対応する。これらは Standard にはない Premium 固有の機能である。選択肢Aの eDiscovery (Standard) は基本的なケース管理・検索・エクスポートは行えるが、機械学習によるレビュー支援やカストディアン管理・特権識別には対応していない。選択肢Cの監査（Standard）はユーザーアクティビティログの検索が目的であり、eDiscovery の証拠収集・レビュープロセスとは異なる。選択肢Bのコンテンツ検索はケース管理機能を持たない単機能の検索ツールであり、訴訟対応の体系的な eDiscovery には不十分である。
