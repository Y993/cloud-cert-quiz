SC-900Microsoft コンプライアンス ソリューションの機能MEDIUM単一選択

ある企業の法務チームが、従業員が組織外の弁護士と交わしたメールの弁護士依頼人特権（Attorney-Client Privilege）を自動識別し、eDiscovery レビューから除外したいと考えている。この機能を提供するのはどちらか。

  1. A. Microsoft Purview eDiscovery (Standard)
  2. B. Microsoft Purview コミュニケーションコンプライアンス
  3. C. Microsoft Purview eDiscovery (Premium)
  4. D. Microsoft Purview 監査（Premium）
解答と解説を見る

正解: C

eDiscovery (Premium) の弁護士依頼人特権検出機能は、レビューセット内のドキュメントを機械学習で分析し、弁護士として登録されたユーザーが関与しているコミュニケーションや法的特権の言語パターンを自動識別してタグ付けする。これにより法的特権のある文書を証拠として不適切に開示するリスクを低減できる。選択肢Aの eDiscovery (Standard) はこの機械学習ベースの特権識別機能を持たず、手動でのレビューに依存する。選択肢Bのコミュニケーションコンプライアンスは規制違反の通信監視が目的であり、eDiscovery の特権識別機能は含まない。選択肢Dの監査（Premium）は監査ログの保持・検索に特化しており、eDiscovery レビュー機能は提供しない。
