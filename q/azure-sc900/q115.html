SC-900Microsoft セキュリティ ソリューションの機能HARD複数選択

ある企業が Microsoft Copilot for Security を評価しています。Microsoft Copilot for Security の正しい説明として正しいものを2つ選択してください。

  1. A. GDPR や ISO 27001 の認証審査を自動的に通過させるコンプライアンス自動化ツールである
  2. B. 自然言語でインシデントの要約・スクリプト解析・対応手順の提案などをセキュリティアナリストに提供する
  3. C. Microsoft Sentinel や Defender XDR など複数の Microsoft セキュリティ製品と統合して利用できる
  4. D. ウイルス対策シグネチャの更新を自動化するためのツールである
  5. E. オンプレミスのサーバーに単独でインストールして使用するスタンドアロン製品である
解答と解説を見る

正解: B, C

Microsoft Copilot for Security は、(B)生成 AI を活用してアナリストが自然言語でインシデント要約・スクリプト解析・次のアクション提案を受けられる AI アシスタントであり、(C)Microsoft Sentinel・Defender XDR・Intune・Purview などの複数の Microsoft セキュリティ製品と統合して使用できます。Eはクラウドサービスであり、オンプレミス単独インストール製品ではないため誤りです。Dはウイルス対策シグネチャ更新ツールの説明であり Copilot for Security の機能ではありません。AはGDPR や ISO 認証を自動通過させる機能は持たず、コンプライアンス自動化ツールという説明も誤りです。

📚 関連サービスの解説: Microsoft Sentinel ・ Microsoft Defender XDR
