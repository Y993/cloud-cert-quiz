SC-900Microsoft コンプライアンス ソリューションの機能MEDIUM単一選択

ある企業が Microsoft Purview eDiscovery (Premium) のレビューセットを使用して、大量のドキュメントを効率的にレビューしたいと考えている。eDiscovery (Premium) のレビューセットで使用できる機能として正しいものはどれか。

  1. A. Azure Blob Storage へのコンテンツの自動アーカイブと圧縮
  2. B. Microsoft Teams の会議録画から字幕テキストの自動生成
  3. C. リアルタイムの機密情報スキャンと SharePoint への自動削除アクション
  4. D. AIによる関連性スコアリング（Relevance）、重複ドキュメントの検出（Near-Duplicate）、スレッドのまとめ（Email Threading）
解答と解説を見る

正解: D

eDiscovery (Premium) のレビューセットには、大量ドキュメントのレビュー効率を高めるための高度な機械学習機能が含まれる。AI による関連性スコアリング（Relevance）は文書のケースへの関連度を自動評価してレビューの優先度付けを行い、ニアデュプリケート検出は類似した内容のドキュメントをグループ化して重複レビューを削減し、メールスレッドのまとめはメールチェーンを再構成して最新のメールのみをレビューすれば前の内容も把握できるようにする。これらにより大量のレビュー対象を効率的に処理できる。選択肢Cの自動削除アクションはレビューセットの機能ではなく、DLP や保持ポリシーが担う。選択肢Aの Azure Blob Storage へのアーカイブはレビューセットの機能ではない。選択肢BのTeams 録画の字幕生成も eDiscovery の機能ではない。
