大規模エンタープライズが Google Cloud を導入する際、Cloud Identity を使うことの主なメリットとして最も適切な説明はどれですか？

  1. A. 社員が個人の Gmail アカウントではなく会社管理のアカウントで Google Cloud を利用できる
  2. B. Google Cloud のサービスクォータが自動的に増加する
  3. C. Google Cloud の請求コストを削減できる
  4. D. Google Cloud リソースの処理性能が向上する
解答と解説を見る

正解: A

Cloud Identity は、Google Workspace（G Suite）の有償ライセンスなしに、会社のドメインでユーザーアカウントとグループを管理できるサービスです。社員が個人の Google アカウントではなく、会社管理のアカウント（例: user@example.com）で Google Cloud にアクセスできるようになり、アカウント管理・セキュリティポリシー・SSO との統合が可能になります。Cloud Identity は請求コストや処理性能に直接影響しません。サービスクォータは Cloud Identity の設定とは独立して管理されます。
