CLF-C02セキュリティとコンプライアンスEASY単一選択

ある企業が、AWSで使用するサービスのコンプライアンス対応状況（GDPR・ISO27001・SOC2等）を確認したいと考えています。AWSが特定の規制・標準に対してどのコンプライアンス認証を持っているかを確認できる最も適切なリソースはどれですか？

  1. A. AWS Artifact
  2. B. Amazon CloudWatch ダッシュボード
  3. C. AWS Config準拠レポート
  4. D. AWS Trusted Advisor
解答と解説を見る

正解: A

AWS ArtifactはAWSのコンプライアンスレポート（SOC 1/2/3・ISO 27001・PCI DSS AOC・FedRAMP等）をオンデマンドでダウンロードできるセルフサービスポータルです。自社の監査対応・コンプライアンス確認に活用できます。②CloudWatchはシステム監視ダッシュボードです。④Trusted AdvisorはAWS環境のベストプラクティスチェックツールです。③AWS ConfigはAWSリソース設定の準拠評価で、AWS自身のコンプライアンス認証文書の提供ではありません。
