CLF-C02クラウドのコンセプトMEDIUM単一選択

あるグローバル企業がAWSを使って低レイテンシーで世界中のユーザーにコンテンツを届けたいと考えています。AWSのグローバルインフラ設計で、このニーズに最も関係する概念はどれですか？

  1. A. VPC（仮想プライベートクラウド）
  2. B. Amazon CloudFrontのエッジロケーション
  3. C. AWSリージョンとアベイラビリティゾーン（AZ）
  4. D. AWS IAMロール
解答と解説を見る

正解: B

エッジロケーションはCloudFrontのCDNネットワークを構成する拠点で、世界中のユーザーに最も近い場所からコンテンツを配信し、レイテンシーを最小化します。③リージョンとAZは可用性・障害分離の設計に使いますが、エンドユーザーへの近接配信の主役ではありません。①VPCはネットワーク分離の仕組みです。④IAMロールはアクセス制御の話です。
