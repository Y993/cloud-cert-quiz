CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、マルチクラウド・ハイブリッドクラウド環境でAWSとオンプレミスのリソースを一元的に管理したいと考えています。オンプレミスのサーバーをAWS Systems Managerで管理したり、AWSの一部サービスをオンプレミスで稼働させたりできる機能を提供するAWSサービスはどれですか？

  1. A. AWS Hybrid Activations (Systems Manager)
  2. B. AWS Direct Connect
  3. C. Amazon EKS Anywhere
  4. D. AWS Outposts
解答と解説を見る

正解: D

AWS Outpostsは、AWSのインフラストラクチャ・サービス（EC2・EBS・RDS・ECS等）をオンプレミスのデータセンターや施設に物理的に設置できるサービスで、AWS管理のラックが物理的にオンプレミスに配置されます。レイテンシー要件や規制によりデータがオンプレミスに必要な場合でもAWSと一元管理できます。③EKS AnywhereはKubernetesをオンプレミスで動かすオプションです。①Hybrid ActivationsはオンプレミスサーバーをSSMで管理する機能ですが、AWSサービス自体のオンプレミス配備ではありません。②Direct Connectは専用線接続です。
