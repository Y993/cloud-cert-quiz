CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、AWS上でKubernetesを使ってコンテナを管理したいと考えています。Kubernetesコントロールプレーンの管理をAWSに委ねることができるマネージドKubernetesサービスはどれですか？

  1. A. Amazon ECS
  2. B. AWS Fargate単体
  3. C. Amazon EKS（Elastic Kubernetes Service）
  4. D. AWS App Runner
解答と解説を見る

正解: C

Amazon EKSはKubernetesコントロールプレーン（etcd、APIサーバー等）のプロビジョニング・更新・スケーリングをAWSが管理するマネージドKubernetesサービスです。Kubernetesの標準APIを使いながら、AWSのセキュリティ・ネットワーキング・ストレージとの統合が可能です。①ECSはAWS独自のコンテナオーケストレーションサービスです（Kubernetes非対応）。②Fargateはコンテナ実行環境で、ECSまたはEKSと組み合わせて使います。④App RunnerはコンテナやコードからWebサービスを手軽にデプロイできるサービスです。
