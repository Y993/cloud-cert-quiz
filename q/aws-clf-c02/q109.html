CLF-C02請求、料金、およびサポートEASY単一選択

ある企業が、AWSのサービス利用料金の見積もりを作成してステークホルダーに提示したいと考えています。AWSのサービス料金を事前に試算するためのツールはどれですか？

  1. A. AWS Cost Explorer
  2. B. AWS Trusted Advisor
  3. C. AWS Budgets
  4. D. AWS Pricing Calculator
解答と解説を見る

正解: D

AWS Pricing Calculatorは、利用予定のAWSサービス・リージョン・利用量を入力してコストを事前に見積もるツールです。シナリオ比較や見積書のエクスポートが可能で、AWSへの移行コスト試算に使われます。①Cost Explorerは実際に発生したコストの分析ツールです。③Budgetsは稼働中の予算管理ツールです。②Trusted Advisorは既存環境のベストプラクティスチェックツールです。いずれも事前の料金見積もりはできません。
