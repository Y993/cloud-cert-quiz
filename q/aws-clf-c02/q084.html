CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、社内のSAP ERPシステムをAWSに移行する計画を立てています。高いI/Oパフォーマンスを必要とするデータベースをEC2インスタンス上で実行するために最も適切なストレージサービスはどれですか？

  1. A. Amazon EFS
  2. B. Amazon EBS（io2 Block Expressボリューム）
  3. C. Amazon S3 Glacier
  4. D. Amazon S3
解答と解説を見る

正解: B

Amazon EBS io2 Block Expressは最高レベルのブロックストレージパフォーマンスを提供し、最大256,000 IOPSと4,000 MB/sのスループットをサポートします。SAP ERPのような高I/Oの本番データベースに最適です。④S3はオブジェクトストレージで、ブロックI/Oが必要なデータベースには使用できません。①EFSはネットワークファイルシステムで、データベースの高いランダムI/Oパフォーマンスには不向きです。③S3 Glacierは低コストの長期アーカイブ用途で、アクセス速度が非常に遅いです。
