CLF-C02クラウドテクノロジーとサービスHARD複数選択

ある企業が、AWS上のWebアプリケーションのパフォーマンス問題を調査しています。アプリケーションのボトルネックを特定するために有用なAWSサービスを2つ選択してください。

  1. A. AWS CloudTrail
  2. B. Amazon Inspector
  3. C. Amazon CloudWatch（メトリクス・ログの監視）
  4. D. AWS X-Ray（分散トレーシング）
解答と解説を見る

正解: C, D

パフォーマンス調査に有用なサービス：③CloudWatchはEC2・RDS・Lambda等のメトリクス（CPU・メモリ・レイテンシー等）とログを収集・可視化し、どのリソースがボトルネックかを特定できます。④AWS X-Rayはマイクロサービス/サーバーレスアーキテクチャの分散トレーシングサービスで、リクエストが各サービスで何ミリ秒費やしているかを可視化し、パフォーマンスのボトルネックを特定できます。①CloudTrailはAPI操作ログで、パフォーマンス分析向けではありません。②Inspectorは脆弱性スキャンです。
