CLF-C02クラウドテクノロジーとサービスEASY単一選択

ある企業が、Webアプリケーションのバックエンドで仮想サーバーを必要としています。OSの選択、ミドルウェアのインストール、セキュリティパッチの適用を自社で管理する必要があります。最も適切なAWSサービスはどれですか？

  1. A. AWS Lambda
  2. B. Amazon S3
  3. C. Amazon RDS
  4. D. Amazon EC2
解答と解説を見る

正解: D

Amazon EC2（Elastic Compute Cloud）はクラウド上の仮想サーバーサービスです。OSの選択（Amazon Linux・Windows Server・Ubuntu等）、ミドルウェアのインストール、セキュリティパッチ管理など、OSレベル以上の設定を顧客が完全にコントロールできます。①Lambdaはサーバーレスコンピューティングで、OS管理は不要ですがコードのみを実行します。③RDSはマネージドデータベースです。②S3はオブジェクトストレージで、サーバーの代替にはなりません。
