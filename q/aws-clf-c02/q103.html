CLF-C02クラウドテクノロジーとサービスMEDIUM単一選択

ある企業が、複数のEC2インスタンスからファイルを共有して読み書きできる共有ファイルシステムが必要です。NFSプロトコルに対応した、複数インスタンスから同時にマウントできるマネージドファイルストレージはどれですか？

  1. A. Amazon S3
  2. B. Amazon EBS（gp3ボリューム）
  3. C. AWS Storage Gateway
  4. D. Amazon EFS（Elastic File System）
解答と解説を見る

正解: D

Amazon EFSはNFSv4プロトコル対応のフルマネージド共有ファイルシステムで、複数のEC2インスタンス（異なるAZも可能）から同時にマウントして読み書きできます。ストレージ容量は自動的にスケールアップ/ダウンします。②EBSは単一のEC2インスタンスにのみアタッチ可能（io2 Multi-Attachは例外的な利用）です。①S3はAPIベースのオブジェクトストレージで、NFSマウントはできません（S3 Mountpointは特殊用途）。③Storage GatewayはオンプレミスとAWSのハイブリッドストレージ接続サービスです。
