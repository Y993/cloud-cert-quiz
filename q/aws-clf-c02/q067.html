CLF-C02クラウドテクノロジーとサービスEASY単一選択

ある企業が、大量の非構造化ファイル（画像・動画・ログファイル）を低コストで長期保存したいと考えています。最も適切なAWSストレージサービスはどれですか？

  1. A. Amazon EBS（Elastic Block Store）
  2. B. Amazon EFS（Elastic File System）
  3. C. Amazon S3
  4. D. Amazon RDS
解答と解説を見る

正解: C

Amazon S3はオブジェクトストレージサービスで、画像・動画・ドキュメント・ログなど非構造化データの大量保存に最適です。99.999999999%（イレブンナイン）の高い耐久性を持ち、S3 Glacierに移行すれば超低コストの長期アーカイブも可能です。①EBSはEC2インスタンスにアタッチするブロックストレージで、大量の非構造化データの保存には適しません。②EFSは複数EC2インスタンスからマウントできるファイルストレージで、長期アーカイブには高コストです。④RDSはリレーショナルデータベースです。
