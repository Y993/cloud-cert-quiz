CLF-C02請求、料金、およびサポートMEDIUM複数選択

ある企業が、AWS Marketplace について調査しています。AWS Marketplaceの特徴として正しいものを2つ選択してください。

  1. A. AWS Marketplaceの使用は、エンタープライズサポートプランの契約者のみ可能
  2. B. サードパーティのソフトウェア（セキュリティ・データベース・機械学習ツール等）をワンクリックでAW上にデプロイできる
  3. C. AWSが開発したソフトウェアのみを販売するプラットフォームである
  4. D. AWS Marketplaceで購入したソフトウェアの料金はAWS請求書に統合して支払いできる
解答と解説を見る

正解: B, D

AWS Marketplaceの特徴：②サードパーティベンダー（Cisco・Trend Micro・Splunk等）が提供するソフトウェア・SaaS・機械学習モデルなどをカタログから選択し、AMIやコンテナとして簡単にデプロイできます。④Marketplaceで購入した製品の料金はAWSの月次請求書に統合されるため、支払い先の管理が一元化されます。③AWSのサービス自体はMarketplaceではなくAWSコンソールから使用します。①すべてのAWSアカウントがMarketplaceにアクセスできます。
