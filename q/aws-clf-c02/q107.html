CLF-C02請求、料金、およびサポートMEDIUM単一選択

ある企業が、AWSの月次請求書を分析して、どのサービス・チーム・プロジェクトにコストがかかっているかを詳細に可視化したいと考えています。最も適切なAWSツールはどれですか？

  1. A. AWS Pricing Calculator
  2. B. Amazon CloudWatch
  3. C. AWS Cost Explorer
  4. D. AWS Budgets
解答と解説を見る

正解: C

AWS Cost Explorerはインタラクティブなコスト・使用量の分析ツールで、サービス別・リージョン別・タグ別・アカウント別の詳細な使用量とコストをグラフで可視化し、過去13ヶ月のデータを分析できます。リザーブドインスタンスの推奨事項も提供します。④Budgetsは予算設定と通知が主目的で、詳細な分析機能はCost Explorerが優れています。①Pricing Calculatorは将来のコスト見積もりツールで、現在の請求分析はできません。②CloudWatchはシステムメトリクスの監視で、コスト分析はできません。
