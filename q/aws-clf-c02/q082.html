CLF-C02クラウドテクノロジーとサービスMEDIUM単一選択

ある企業が、AWSのデータベースサービスを選定しています。ユーザーのセッション情報や一時的なキャッシュデータを高速に読み書きするために、インメモリデータストアが必要です。最も適切なAWSサービスはどれですか？

  1. A. Amazon ElastiCache
  2. B. Amazon Redshift
  3. C. Amazon RDS（MySQL）
  4. D. Amazon DynamoDB
解答と解説を見る

正解: A

Amazon ElastiCacheはMemcachedまたはRedisのマネージドインメモリキャッシュサービスです。ディスクではなくメモリにデータを保存するため、マイクロ秒〜ミリ秒単位の超高速な読み書きが可能で、セッション管理・データベースクエリキャッシュ・リアルタイムリーダーボードなどに最適です。③RDSはディスクベースのリレーショナルDBです。④DynamoDBは高速なNoSQLですが、インメモリキャッシュ専用ではありません。②Redshiftはデータウェアハウス（分析）用途のサービスです。
