CLF-C02請求、料金、およびサポートHARD単一選択

ある企業が、AWSコストの最適化を進めています。EC2インスタンスが過剰スペックである可能性を検出し、より適切なインスタンスサイズを推奨する機能を持つAWSサービスはどれですか？

  1. A. Amazon CloudWatch
  2. B. AWS Compute Optimizer
  3. C. AWS Cost Explorer のリザーブドインスタンス推奨
  4. D. AWS Budgets
解答と解説を見る

正解: B

AWS Compute Optimizerは機械学習を使用してEC2・Lambda・EBS・ECSの実際の使用状況を分析し、適切なサイズへの変更（右サイジング）を推奨するサービスです。過剰スペックのインスタンスや過小スペックのインスタンスを特定し、最適なインスタンスタイプを提案します。①CloudWatchはメトリクス収集・監視ですが、インスタンス推奨機能はありません。④Budgetsは予算管理です。③Cost ExplorerのRI推奨はリザーブドインスタンスの購入最適化ですが、インスタンスサイズの見直し推奨はCompute Optimizerが専門です。
