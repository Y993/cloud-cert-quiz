CLF-C02請求、料金、およびサポートMEDIUM単一選択

ある企業が、複数のAWSアカウントを持つ子会社・部門の請求を一元管理し、月末の支払いをまとめて行いたいと考えています。最も適切なAWS機能はどれですか？

  1. A. AWS Organizationsの一括請求（Consolidated Billing）
  2. B. AWS Cost Explorer
  3. C. AWS Budgets
  4. D. AWS Marketplace
解答と解説を見る

正解: A

AWS Organizationsの一括請求（Consolidated Billing）機能により、組織内の複数のAWSアカウントの使用量を1つの管理アカウントにまとめて請求できます。各アカウントの使用量を合算することで、ボリュームディスカウントの適用基準に達しやすくなるメリットもあります。③②は個別のコスト管理・分析ツールです。④AWS Marketplaceはサードパーティソフトウェアの販売プラットフォームです。
