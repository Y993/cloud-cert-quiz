CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、IoTデバイスから送られてくるリアルタイムのストリームデータ（毎秒数千件）を収集・処理・分析したいと考えています。最も適切なAWSサービスはどれですか？

  1. A. Amazon RDS
  2. B. AWS Batch
  3. C. Amazon Kinesis Data Streams
  4. D. Amazon SQS
解答と解説を見る

正解: C

Amazon Kinesis Data Streamsはリアルタイムのデータストリームを大規模に収集・処理するサービスです。数千件/秒のデータを低レイテンシーで取り込み、Lambda・Kinesis Data Analytics・Redshiftなどにリアルタイムで配信できます。④SQSはメッセージキューで、リアルタイムストリーム処理には適していません（保持期間・スループットに制限）。①RDSはリレーショナルDBで高頻度のストリームデータ取り込みには向きません。②Batchはバッチ処理向けで、リアルタイム処理ではありません。

📚 関連サービスの解説: Amazon Kinesis
