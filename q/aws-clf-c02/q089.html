CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、毎月数テラバイトの販売データを分析してビジネスインサイトを得たいと考えています。大量のデータをSQLで高速に分析するためのデータウェアハウスサービスとして最も適切なものはどれですか？

  1. A. Amazon DynamoDB
  2. B. Amazon Redshift
  3. C. Amazon ElastiCache
  4. D. Amazon RDS（PostgreSQL）
解答と解説を見る

正解: B

Amazon Redshiftはペタバイト規模のデータウェアハウスサービスで、大量の構造化データに対するOLAP（オンライン分析処理）クエリを高速に実行できます。列指向ストレージとMPP（Massively Parallel Processing）アーキテクチャにより、複雑な集計クエリに最適化されています。④RDSはトランザクション処理（OLTP）向けで、大規模分析には性能が劣ります。①DynamoDBはNoSQLで複雑なSQLアナリティクスには不向きです。③ElastiCacheはインメモリキャッシュでデータ分析用ではありません。
