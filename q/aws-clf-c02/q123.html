CLF-C02請求、料金、およびサポートEASY単一選択

ある企業が、AWSのコスト削減に向けたアドバイスを受けたいと考えています。コスト最適化・セキュリティ・パフォーマンス・耐障害性・サービス制限の5カテゴリでAWS環境のベストプラクティス準拠状況を確認できるAWSサービスはどれですか？

  1. A. Amazon CloudWatch
  2. B. AWS Cost Explorer
  3. C. AWS Trusted Advisor
  4. D. AWS Budgets
解答と解説を見る

正解: C

AWS Trusted Advisorはコスト最適化・セキュリティ・パフォーマンス・耐障害性・サービス制限（クォータ）の5つのカテゴリでAWS環境を自動チェックし、改善推奨事項を提供します。サポートプランに応じてチェック項目数が異なります（ビジネス以上でフルアクセス）。②Cost Explorerはコスト分析ツールです。①CloudWatchはメトリクス・ログ監視です。④Budgetsは予算設定と通知です。いずれも5カテゴリにわたるベストプラクティスの包括的なチェックはしません。
