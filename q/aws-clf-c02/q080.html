CLF-C02クラウドテクノロジーとサービスEASY単一選択

ある企業が、アプリケーションのコードをサーバーなしにデプロイして実行したいと考えています。イベント（HTTPリクエスト、S3のオブジェクト作成、SQSメッセージ等）に応じてコードを実行でき、サーバー管理が不要なサービスはどれですか？

  1. A. Amazon ECS（Elastic Container Service）
  2. B. AWS Lambda
  3. C. Amazon EC2
  4. D. Amazon Lightsail
解答と解説を見る

正解: B

AWS Lambdaはサーバーレスコンピューティングサービスで、サーバーのプロビジョニング・管理なしにコードをアップロードするだけで、各種イベントをトリガーに実行できます。実際の実行時間とリクエスト数に応じた従量課金で、アイドル時は料金が発生しません。③EC2は仮想サーバーでOSの管理が必要です。①ECSはコンテナのオーケストレーションサービスです。④Lightsailはシンプルな仮想プライベートサーバーサービスで、サーバー管理は依然として必要です。
