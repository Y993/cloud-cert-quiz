CLF-C02クラウドテクノロジーとサービスHARD単一選択

ある企業が、ペタバイト規模のデータをオンプレミスからAWSに移行する必要があります。利用可能なインターネット帯域が低く、オンラインでの転送には数年かかると試算されました。最も適切なAWSデータ移行サービスはどれですか？

  1. A. AWS Direct Connect
  2. B. Amazon S3 Transfer Acceleration
  3. C. AWS DataSync
  4. D. AWS Snowball Edge
解答と解説を見る

正解: D

AWS Snowball Edgeはスーツケースサイズの物理デバイスで、100TBまたは80TBのストレージ容量を持ちます。オンプレミスでデータをロードしてAWSに配送し、AWSがS3にインポートします。ネットワーク転送が現実的でない大規模データ移行に最適です。③DataSyncはネットワーク経由の転送に特化しており、ペタバイト規模には実用的でありません。②Transfer Accelerationはネットワーク転送の高速化で、根本的な帯域不足の解決にはなりません。①Direct Connectは専用線を引くのに時間とコストがかかります。
