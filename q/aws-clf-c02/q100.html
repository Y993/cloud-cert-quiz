CLF-C02クラウドテクノロジーとサービスMEDIUM単一選択

ある企業が、Webアプリケーションのパフォーマンスを監視し、エラーレートが5%を超えた際に担当者にメール通知を送りたいと考えています。最も適切なAWSサービスの組み合わせはどれですか？

  1. A. AWS CloudTrail + Amazon SES
  2. B. AWS Config + AWS Lambda
  3. C. Amazon CloudWatch Alarms + Amazon SNS
  4. D. Amazon Inspector + Amazon SQS
解答と解説を見る

正解: C

Amazon CloudWatch Alarmsは設定したメトリクス（エラーレート等）が閾値を超えた際にアラームを発行し、Amazon SNSのトピックを通じてメール・SMS・Lambda等に通知できます。この組み合わせはAWSのモニタリング・通知の最も基本的なパターンです。①CloudTrailはAPI操作ログで、アプリエラー率のモニタリングはしません。④Inspectorは脆弱性スキャン、SQSはメッセージキューで組み合わせが不適切です。②ConfigはAWSリソース設定の追跡で、アプリのエラー率監視はしません。
