AI-900自然言語処理ワークロードMEDIUM複数選択

ある企業が Azure AI Language を使った NLP パイプラインを設計している。テキストデータに対して実行できる機能として、Azure AI Language が提供するものを2つ選択してください。

  1. A. 感情分析：テキストのポジティブ・ネガティブ・中立スコアを返す
  2. B. 顔認識：テキスト内に記載された人名から顔写真を検索する
  3. C. 固有表現認識（NER）：テキスト中の人名・組織・日時などを識別する
  4. D. オブジェクト検出：テキスト内に含まれる画像内の物体を識別する
  5. E. 音声エンコーディング：テキストを音声波形データに変換する
解答と解説を見る

正解: A, C

Azure AI Language が提供する代表的なテキスト分析機能として、感情分析（Sentiment Analysis）と固有表現認識（NER）が正解である。感情分析はポジティブ・ネガティブ・中立を数値スコアで返し、NER は人名・組織名・日時・場所などのエンティティをカテゴリ付きで抽出する。選択肢Dのオブジェクト検出は Azure AI Vision の機能（画像解析）であり、Azure AI Language の機能ではない。選択肢Bの顔認識は Azure AI Vision / Azure AI Face の機能であり、テキスト処理サービスではない。選択肢Eの音声エンコーディングは Azure AI Speech の領域であり、Azure AI Language ではテキスト→音声変換は提供しない。
