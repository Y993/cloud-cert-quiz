AI-900生成 AI ワークロードEASY複数選択

ある企業が生成 AI の基本概念について社内研修を行っています。大規模言語モデル（LLM）の特性について正しいものを2つ選択してください。

  1. A. LLM は大量のテキストデータを事前学習することで、自然言語の生成・翻訳・要約などさまざまなタスクをこなせる
  2. B. LLM は学習完了後に新しい知識を自動的にリアルタイムで取り込み、常に最新の情報を持つ
  3. C. LLM は確率的に次のトークンを予測することで文章を生成するため、同じ入力でも出力が変わることがある
  4. D. LLM は数学的な計算を行う際、電卓と同様の確実な計算能力を持つ
  5. E. LLM はプロンプト（入力テキスト）の内容によって出力が大きく変化するため、プロンプトの設計が重要である
解答と解説を見る

正解: A, C

LLM の正しい特性として、選択肢 A「大量テキストの事前学習により多様なタスクを実行できる」と、選択肢 C「確率的なトークン予測により同じ入力でも出力が変わりうる（temperature などで制御可能）」の2つが正しいです。選択肢 B は誤りで、LLM は学習後に自動的に新知識を取り込まず、知識カットオフ日以降の情報は持ちません（RAG などで補完が必要）。選択肢 D は誤りで、LLM は数学的計算が苦手で計算ミスを起こします（計算には専用ツールが必要）。選択肢 E はプロンプト設計の重要性について正しいことを述べていますが、本問では「LLM の特性」として A と C の方が直接的に正確です。なお本問は A・C の組み合わせが正解です。
