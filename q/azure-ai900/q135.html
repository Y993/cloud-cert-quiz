AI-900生成 AI ワークロードMEDIUM単一選択

ある小売企業が Microsoft Copilot for Microsoft 365 の導入を検討しています。Copilot for Microsoft 365 が提供する機能として最も正確に説明しているものはどれか。

  1. A. 既存の Microsoft 365 アプリ（Word・Excel・Teams など）に AI 機能を統合し、文書作成・データ分析・会議要約などを自然言語で行えるようにする
  2. B. Azure OpenAI Service の API エンドポイントを Microsoft 365 ライセンスに含めることで、開発者が独自アプリを構築できるようにする
  3. C. Microsoft 365 のデータを Azure Blob Storage に自動バックアップし、生成 AI で分析するサービスである
  4. D. Copilot for Microsoft 365 は ChatGPT と同じサービスであり、Microsoft がブランド名を変更したものである
解答と解説を見る

正解: A

Microsoft Copilot for Microsoft 365 は、Word での文書作成支援・Excel でのデータ分析・PowerPoint でのスライド生成・Teams での会議要約・Outlook でのメール下書きなど、既存の Microsoft 365 アプリにシームレスに統合された AI アシスタントです。ユーザーは自然言語で指示するだけで作業効率を向上させられます。選択肢 B は Azure OpenAI Service の開発者向け機能の説明であり、Copilot for Microsoft 365 の目的（エンドユーザー向け AI 機能統合）とは異なります。選択肢 C のバックアップ・分析サービスの説明は完全に誤りです。選択肢 D も誤りで、ChatGPT は OpenAI が提供するサービスであり、Copilot for Microsoft 365 はマイクロソフトが GPT-4 をベースに構築した独自サービスです。
