AI-900機械学習の基本原則EASY単一選択

あるスタートアップの機械学習エンジニアが、適切な機械学習アルゴリズムを選択するために最適なアルゴリズムを自動で見つけたいと考えている。複数のアルゴリズムとハイパーパラメータの組み合わせを自動的に試してくれるAzure Machine Learningの機能はどれか。

  1. A. Azure Machine Learning のコンピューティングインスタンス
  2. B. Azure Machine Learning のエンドポイント
  3. C. Azure Machine Learning のデザイナー
  4. D. Azure Machine Learning の自動ML（AutoML）
解答と解説を見る

正解: D

自動ML（AutoML）はアルゴリズムの選択・特徴量エンジニアリング・ハイパーパラメータのチューニングを自動化する機能であり、試行錯誤を最小化して最適なモデルを見つけたい場合に最も適している。Cのデザイナーはビジュアルなパイプライン構築ツールであり、アルゴリズムをユーザーが選んで接続するもので自動選択は行わない。Bのエンドポイントは学習済みモデルをデプロイして推論APIとして公開する機能であり、学習段階のアルゴリズム選択とは別の役割。Aのコンピューティングインスタンスは開発用の仮想マシンであり、アルゴリズム探索機能ではない。
