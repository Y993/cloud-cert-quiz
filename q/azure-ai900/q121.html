AI-900生成 AI ワークロードEASY単一選択

ある医療系スタートアップが、患者向けの症状チェックチャットボットを Azure OpenAI Service で構築しています。出力に不適切または有害なコンテンツが含まれないようにするため、Azure が提供する組み込みのセーフガードとして最も直接的に機能するものはどれか。

  1. A. Azure OpenAI コンテンツフィルター
  2. B. Azure Defender for Cloud
  3. C. Azure Policy
  4. D. Azure Private Link
解答と解説を見る

正解: A

Azure OpenAI コンテンツフィルターは、プロンプトおよびモデルの出力に対して、ヘイトスピーチ・性的コンテンツ・暴力・自傷などのカテゴリを自動的に検出しブロックするセーフガードです。医療系アプリケーションなど高リスク用途では特にデフォルト設定の確認と必要に応じた厳格化が重要です。選択肢 D の Azure Private Link はネットワーク隔離のためのサービスであり、コンテンツ安全とは無関係です。選択肢 C の Azure Policy はリソース構成のコンプライアンス管理ツールです。選択肢 B の Azure Defender for Cloud はクラウドセキュリティ態勢管理（CSPM）と脅威検知のサービスで、AI コンテンツ安全とは役割が異なります。
