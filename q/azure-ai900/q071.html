AI-900コンピュータ ビジョン ワークロードMEDIUM単一選択

ある動画配信サービスが、アップロードされた動画コンテンツのメタデータを自動生成したい。各動画から「登場人物の顔」「シーンの場所」「表示されているテキスト（字幕・看板）」「感情」などを自動抽出し、検索用インデックスに格納したい。最もこの要件に適した Azure サービスはどれか。

  1. A. Azure AI Custom Vision で動画の各フレームを分類する
  2. B. Azure AI Face のみを使用し、登場人物の顔情報だけを抽出する
  3. C. Azure Video Indexer：動画から顔・音声・テキスト・感情などを包括的に分析できる
  4. D. Azure AI Vision の Read API のみを使用し、動画フレームからテキストだけを抽出する
解答と解説を見る

正解: C

Azure Video Indexerは動画コンテンツに特化したAI分析サービスであり、顔の検出と識別・音声の文字起こし・翻訳・表示テキストのOCR・シーン検出・感情分析・キーフレーム抽出など、本シナリオが求める機能をすべてカバーする。Read APIのみでは動画の顔・感情・場所情報は抽出できずテキストのみとなり要件を部分的にしか満たさない。Custom Visionは動画分析に特化していなく、フレーム単位の分類に留まり音声・感情等は取れない。Azure AI Faceのみでは顔以外の情報（テキスト・シーン・感情・音声等）が得られず、要件を部分的にしか満たさない。
