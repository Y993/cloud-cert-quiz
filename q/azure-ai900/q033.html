AI-900機械学習の基本原則EASY単一選択

ある企業がAzure Machine Learningを使って機械学習パイプラインを構築したいと考えている。データサイエンティストがPythonコードをほとんど書かず、グラフィカルなドラッグ＆ドロップ操作でモデルを構築できる機能はどれか。

  1. A. Azure Machine Learning のコンピューティングクラスター
  2. B. Azure Machine Learning の自動ML（AutoML）
  3. C. Azure Machine Learning のデータストア
  4. D. Azure Machine Learning のデザイナー
解答と解説を見る

正解: D

Azure Machine Learningの「デザイナー」はグラフィカルなドラッグ＆ドロップインターフェースで機械学習パイプラインを構築できる機能であり、コーディングを最小化したい場合に適している。Bの自動ML（AutoML）はアルゴリズム選択・ハイパーパラメータ調整を自動化する機能であり、操作はウィザード形式だがドラッグ＆ドロップでパイプラインを組む機能ではない。Aのコンピューティングクラスターはモデル訓練を実行するための計算リソースであり、UI操作の仕組みとは異なる概念。Cのデータストアはデータソースへの接続情報を管理する機能であり、モデル構築UIとは関係しない。
