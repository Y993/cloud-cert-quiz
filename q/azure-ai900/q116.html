AI-900生成 AI ワークロードEASY単一選択

ある企業のマーケティング担当者が、製品紹介ブログ記事の下書きを短時間で作成したいと考えています。担当者はプログラミングの知識がなく、日本語でテキストを入力するだけで利用できるツールを求めています。Microsoft 365 を契約済みの環境で最も適切なアプローチはどれか。

  1. A. Microsoft 365 Copilot を使って Word 上でブログ草稿の生成を依頼する
  2. B. Azure AI Foundry でカスタムモデルをデプロイしてテキストを生成する
  3. C. Azure Machine Learning でファインチューニング済みモデルをトレーニングして利用する
  4. D. Azure OpenAI Service のプレイグラウンドに直接アクセスしてテキストを生成する
解答と解説を見る

正解: A

Microsoft 365 Copilot は Word・Outlook・Teams などの既存 Microsoft 365 アプリに統合された生成 AI アシスタントであり、プログラミング不要で自然言語の指示だけで文書作成や草稿生成が行えるため、非技術者のマーケティング担当者に最適です。選択肢 D の Azure OpenAI プレイグラウンドは開発者向けの実験環境であり、エンタープライズ展開には不向きです。選択肢 B の Azure AI Foundry はモデルのデプロイや管理を行う開発者プラットフォームです。選択肢 C の Azure Machine Learning はモデルトレーニングのためのプラットフォームであり、今回の用途には過剰かつ非現実的な対応です。
