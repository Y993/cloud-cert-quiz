AI-900生成 AI ワークロードMEDIUM単一選択

ある企業が Azure AI Foundry を使って生成 AI ソリューションの開発を始めようとしています。Azure AI Foundry の主な目的として最も正確な説明はどれか。

  1. A. 生成 AI アプリケーションの開発・評価・デプロイ・監視をエンドツーエンドで行うための統合開発プラットフォームである
  2. B. Azure OpenAI Service のモデルのみを管理でき、サードパーティモデルは利用できないサービスである
  3. C. Microsoft Teams や Outlook などのアプリケーションに AI 機能を提供する専用サービスである
  4. D. Azure の仮想マシンや Kubernetes クラスターを管理するインフラ管理ポータルである
解答と解説を見る

正解: A

Azure AI Foundry（旧 Azure AI Studio）は、生成 AI アプリケーションの開発をエンドツーエンドでサポートする統合プラットフォームです。モデルカタログからのモデル選択・プロンプトフローによるアプリ構築・評価・デプロイ・監視まで一貫して行えます。Azure OpenAI モデルだけでなく、Meta Llama・Mistral などのオープンソースモデルも利用可能です。選択肢 D はインフラ管理の説明であり、Azure Portal の機能に近いです。選択肢 C は Microsoft 365 Copilot や Copilot Studio の説明に近く、Azure AI Foundry の目的とは異なります。選択肢 B は誤りで、Azure AI Foundry のモデルカタログには多数のサードパーティおよびオープンソースモデルが含まれています。
