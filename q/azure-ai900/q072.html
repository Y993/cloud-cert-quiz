AI-900コンピュータ ビジョン ワークロードMEDIUM単一選択

ある都市交通局が、主要交差点に設置したカメラのリアルタイム映像を解析して、歩行者が車道に侵入した際に警告を発するシステムを構築したい。リアルタイムの映像ストリームを処理し、歩行者の位置と禁止エリアへの侵入を検知する必要がある。最もこの要件に適した Azure サービスはどれか。

  1. A. Azure AI Custom Vision の物体検出：動画フレームを個別に送信して歩行者を検出する
  2. B. Azure AI Face：カメラ映像から歩行者の顔を認識し通知する
  3. C. Azure AI Vision の空間分析（Spatial Analysis）：ライブカメラ映像から人の動線と指定エリアへの侵入を検知する
  4. D. Azure Video Indexer：録画動画の事後分析でレポートを生成する
解答と解説を見る

正解: C

Azure AI Visionの空間分析（Spatial Analysis）は、カメラのライブ映像ストリームをリアルタイムで処理し、人の位置・動線・指定ゾーンへの侵入・密集度などを検知するために設計されたサービスである。禁止エリアへの歩行者侵入のリアルタイム検知には空間分析が最適である。Custom Visionの物体検出はAPIへの個別フレーム送信が必要で、ライブストリームのリアルタイム処理には向かず遅延が大きい。Azure Video Indexerは録画動画の事後分析向けであり、リアルタイム警告システムには適さない。Azure AI Faceは顔認証に特化しており、歩行者の位置・エリア侵入の検知機能は持たない。
