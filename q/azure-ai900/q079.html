AI-900コンピュータ ビジョン ワークロードMEDIUM単一選択

ある出版社がデジタルアーカイブ化プロジェクトを進めている。数万ページの古い書籍（活字印刷）と著者の手書きメモが混在したドキュメントをスキャンしており、テキスト検索可能なデジタルデータに変換したい。Azure AI Vision の Read API について正しい説明はどれか。

  1. A. PDF形式のファイルは処理できず、JPEG・PNG形式のみ対応している
  2. B. 日本語の認識には対応していないため、別途翻訳APIと組み合わせる必要がある
  3. C. 印刷テキストと手書き文字の両方を1つのAPIで認識でき、複数言語にも対応している
  4. D. 印刷テキストのみ対応しており、手書き文字の認識は別サービスを必要とする
解答と解説を見る

正解: C

Azure AI VisionのRead APIは印刷テキストと手書き文字の両方を1つのAPIで処理でき、多数の言語（日本語を含む）に対応している。スキャンした書籍ページと手書きメモが混在するドキュメントの一括デジタル化に適している。印刷のみという制限はなく手書きにも対応するため選択肢Dは誤り。日本語対応も提供されているため選択肢Bは誤り。PDF形式にも対応しており、JPEG・PNG・TIFF・PDFなど複数の入力形式をサポートしているため選択肢Aも誤りである。
