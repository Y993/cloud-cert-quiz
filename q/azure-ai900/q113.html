AI-900自然言語処理ワークロードMEDIUM複数選択

ある企業がグローバルカスタマーサポートの自動化において、テキストと音声の両チャネルに対応したシステムを構築している。要件として「電話音声を英語テキストへ書き起こす」と「書き起こしたテキストを複数言語に翻訳する」がある。これらの要件にそれぞれ対応する Azure サービスの組み合わせとして正しいものを2つ選択してください。

  1. A. Azure AI Language の CLU：音声を認識してインテントを分類する
  2. B. Azure AI Language の感情分析：テキストを複数言語に翻訳する
  3. C. Azure AI Speech の音声テキスト変換：電話音声を英語テキストに書き起こす
  4. D. Azure AI Language の質問応答：テキストを複数の言語に変換する
  5. E. Azure AI Translator：書き起こした英語テキストを複数言語に翻訳する
解答と解説を見る

正解: C, E

電話音声のテキスト書き起こしには Azure AI Speech の音声テキスト変換（Speech to Text）、テキストの多言語翻訳には Azure AI Translator が適切なサービスである。この2つを組み合わせることで要件を満たすパイプラインを構築できる。選択肢Bの感情分析はテキストの感情スコアを判定する機能であり、翻訳機能は持たない。選択肢AのCLUは音声ではなくテキスト入力に対してインテント分類を行うものであり、音声書き起こし機能は持たない（Speech+CLUの連携は可能だが単独では書き起こしを行わない）。選択肢Dの質問応答は既存ドキュメントからの回答抽出機能であり、多言語翻訳は行わない。
