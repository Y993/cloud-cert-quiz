AI-900コンピュータ ビジョン ワークロードMEDIUM単一選択

ある教育機関が、試験の解答用紙（手書き）を自動採点するシステムを構築したい。解答用紙には手書きの記述回答と選択回答（〇×や記号）が混在している。最もこの要件に適した Azure AI の機能はどれか。

  1. A. Azure AI Vision の画像キャプション生成：解答用紙の内容を文章で説明させる
  2. B. Azure AI Face：解答用紙の記入者の顔を識別する
  3. C. Azure AI Custom Vision の画像分類：解答用紙を「正解」「不正解」に分類する
  4. D. Azure AI Vision の Read API（OCR）：手書き文字を含む印刷・手書きテキストの抽出に対応する
解答と解説を見る

正解: D

Azure AI VisionのRead APIは印刷テキストだけでなく手書き文字の認識にも対応しており、解答用紙のような手書き記述と印刷文字が混在するドキュメントの文字抽出に最適である。多言語対応・複雑なレイアウトへの対応も充実している。画像キャプション生成は「このドキュメントには解答が書かれています」のような自然言語説明を生成するものであり、特定フィールドの採点に使えない。Custom Visionの画像分類は「正解/不正解」の二値判定には使えるが、手書きテキストの内容を読み取る機能はない。Azure AI Faceは人物識別専用で解答内容の読み取りには無関係。
