AI-900コンピュータ ビジョン ワークロードMEDIUM単一選択

ある観光地の管理局が、観光スポットに設置したカメラで1日あたりの訪問者数と混雑ピーク時間帯を計測したい。カメラ映像から人の入退場を自動カウントし、混雑状況のダッシュボードにリアルタイムでデータを送りたい。最もこの要件に適した Azure サービスはどれか。

  1. A. Azure AI Vision の空間分析（Spatial Analysis）：ライブカメラ映像から人のカウント・動線・エリア滞在時間を分析する
  2. B. Azure Video Indexer：録画動画を事後処理して人数をカウントする
  3. C. Azure AI Face：通過した人の顔を全員識別して訪問者数を算出する
  4. D. Azure AI Custom Vision の画像分類：混雑している/していないで分類する
解答と解説を見る

正解: A

Azure AI Visionの空間分析は、ライブカメラ映像をリアルタイムで処理し、指定エリアを通過した人数カウント・人の動線トラッキング・エリア内の滞在時間・密集度などを継続的に計測できる。観光地の訪問者数とピーク時間帯のリアルタイム計測に最適である。Custom Visionの分類は「混雑している/していない」の二値判定に留まり、正確な人数カウントはできない。Video Indexerは録画動画の事後処理向けであり、リアルタイムダッシュボードには不向き。Azure AI Faceによる全訪問者の顔識別は、プライバシー上の問題があるうえに訪問者数カウントとして過剰かつコスト高である。
