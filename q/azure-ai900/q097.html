AI-900自然言語処理ワークロードMEDIUM単一選択

ある E コマース企業が商品ページに「この商品について質問する」チャット機能を追加したい。商品の仕様・使い方・返品ポリシーをまとめた FAQ ドキュメントを知識源として、顧客の質問に自動回答させたい。追加のモデルトレーニングを最小限にしてすぐに展開できる Azure のサービスはどれか。

  1. A. Azure AI Language の会話言語理解（CLU）：カスタムインテントとエンティティを定義してトレーニングが必要
  2. B. Azure AI Language の質問応答（Question Answering）：FAQ ドキュメントを取り込むだけで Q&A ボットを構築できる
  3. C. Azure OpenAI Service：大規模言語モデルを使って自由に応答を生成する
  4. D. Azure AI Language のキーフレーズ抽出：FAQ の重要語句を抽出してインデックス化する
解答と解説を見る

正解: B

Azure AI Language の質問応答（Question Answering）は既存の FAQ ドキュメントや URL を知識源として取り込むだけで動作し、追加のモデルトレーニングを最小限に Q&A ボットを構築できる。ドキュメントから質問・回答ペアを自動抽出し、ユーザーの質問に最も近い回答を返す。選択肢Aの会話言語理解（CLU）はインテント・エンティティを自分で定義しトレーニングデータを用意する必要があり、すぐに展開できるとは言えない。選択肢CのAzure OpenAI Serviceは強力だが初期設定・プロンプト設計・コスト管理が必要であり「最小限」要件に反する。選択肢Dのキーフレーズ抽出は語句の抽出のみで、質問への回答は行わない。
