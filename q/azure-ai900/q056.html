AI-900機械学習の基本原則EASY複数選択

あるデータサイエンティストが、機械学習モデルの構築に使うAzure Machine Learning の主要機能について説明している。「コードを書かずにモデルを構築・評価できる」という特徴を持つ機能を2つ選択してください。

  1. A. Azure ML デザイナー：ドラッグ＆ドロップでパイプラインを構築できるビジュアルツール
  2. B. Azure Databricks：大規模データ処理と機械学習を統合するデータエンジニアリングプラットフォーム
  3. C. 自動ML（AutoML）：アルゴリズムとハイパーパラメータを自動で探索しウィザード形式で操作できる
  4. D. Azure ML SDK（Python）：Pythonコードでモデル訓練・デプロイを制御する開発者向けライブラリ
  5. E. Azure DevOps：CI/CDパイプラインでMLOpsを構築するためのツール
解答と解説を見る

正解: A, C

コードを書かずにモデルを構築できるAzure MLの機能は自動ML（AutoML）とデザイナーの2つである。AutoMLはウィザード形式でデータとタスクを設定するだけで多数のアルゴリズムを自動試行してくれる。デザイナーはビジュアルなドラッグ＆ドロップでデータ前処理からモデル訓練・評価までのパイプラインを構築できる。Dの Azure ML SDK はPythonコードを書く必要があるため「コード不要」には該当しない。Bの Azure Databricks はSparkベースのデータ処理プラットフォームで、コードを書くのが基本の開発者向けツール。EのAzure DevOpsはCI/CDの管理ツールであり、MLモデルの構築・評価ツールではない。
