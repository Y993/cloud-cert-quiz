AZ-104Azure ストレージの実装と管理MEDIUM単一選択

ある開発者が AzCopy を使って Azure BLOB ストレージとオンプレミス環境の間でデータを同期させたい。毎晩バッチ処理で差分のみを転送し、送信元に存在しないファイルを送信先から削除するコマンドとして正しいものはどれか。

  1. A. azcopy copy --recursive
  2. B. azcopy make --sync
  3. C. azcopy sync --delete-destination=true
  4. D. azcopy remove --incremental
解答と解説を見る

正解: C

azcopy sync コマンドは送信元と送信先のファイルを比較して差分のみを転送する増分コピー機能を持ち、--delete-destination=true オプションを指定すると送信元に存在しないファイルを送信先から削除する双方向整合性を実現できる。定期的なバッチ同期に適した設計である。azcopy copy --recursive はファイルのコピーには使えるが増分転送の最適化がなく、送信先の余分なファイル削除機能もない。azcopy make は新しいコンテナや共有を作成するコマンドであり、sync オプションは存在しない。azcopy remove はファイル削除専用コマンドであり、--incremental オプションは存在しない。
