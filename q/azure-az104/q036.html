AZ-104Azure ストレージの実装と管理EASY単一選択

ある製造業の企業がオンプレミスの NAS を Azure に移行したい。SMB プロトコルを使って Windows クライアントおよびサーバーから直接マウントでき、既存のファイル共有アクセスをほぼそのまま維持したい場合に最も適切な Azure サービスはどれか。

  1. A. Azure Files（SMB 共有）
  2. B. Azure Data Lake Storage Gen2
  3. C. Azure Disk Storage（マネージドディスク）
  4. D. Azure BLOB ストレージ（階層型名前空間を有効化）
解答と解説を見る

正解: A

Azure Files は SMB 2.1/3.x プロトコルを完全サポートし、Windows・Linux・macOS クライアントからネットワークドライブとしてマウントできる。既存の NAS 置き換えに最も適しており、Windows の資格情報（AD 統合も可能）でのアクセスを維持できる。Azure BLOB ストレージは REST/HTTPS 経由でアクセスするオブジェクトストレージであり SMB マウントはサポートしない。Azure Disk Storage は VM にアタッチするブロックデバイスであり、複数クライアントによる同時マウント（共有ディスクを除く）には対応しない。Azure Data Lake Storage Gen2 は階層型名前空間付き BLOB ストレージであり、分析ワークロード向けで SMB マウントには適さない。
