AIF-C01責任ある AI に関するガイドラインMEDIUM単一選択

ある企業が、SageMaker で学習した与信スコアリングモデルの「公平性」を評価したいと考えています。SageMaker Clarify と SageMaker Model Monitor の使い分けとして最も正確な説明はどれですか？

  1. A. Clarify はデプロイ後の継続監視に使い、Model Monitor は学習フェーズのバイアス分析に使う
  2. B. Model Monitor はバイアスの検出のみを行い、Clarify はデータドリフトのみを検出する
  3. C. Clarify は学習データや学習済みモデルのバイアス指標と特徴量重要度（SHAP）を分析し、Model Monitor は本番エンドポイントのデータ・予測ドリフトを継続的に監視する
  4. D. 両サービスは完全に同一の機能を持ち、どちらを使っても同じ結果が得られる
解答と解説を見る

正解: C

SageMaker Clarify は学習データの偏り（クラス不均衡など）、学習済みモデルの予測バイアス（DPPL など）、および SHAP ベースの説明可能性分析を提供します。主に学習・評価フェーズで活用します。一方 SageMaker Model Monitor は、本番エンドポイントに流れる入力データの統計的変化（データドリフト）や予測結果の品質劣化をリアルタイムまたはスケジュールで継続監視します。Aはそれぞれの役割を逆に説明しており誤りです。D は両サービスが同一と言っており明らかに誤りです。Bは各サービスの機能を逆に割り当てており誤りです。
