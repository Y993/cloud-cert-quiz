AIF-C01生成 AI の基礎MEDIUM複数選択

ある企業が Amazon Bedrock を用いた生成 AI アプリケーションを本番環境に展開する際、出力品質とリスク管理のために導入すべき対策を2つ選択してください。

  1. A. すべての API 呼び出しを同期的に単一スレッドで処理し、並行リクエストを禁止する
  2. B. モデルの temperature を常に 2.0 以上に設定し、出力の多様性を最大化する
  3. C. 本番リリース前に人間による評価（Human Evaluation）を実施し、出力品質・有害性・バイアスを確認する
  4. D. Amazon Bedrock Guardrails でコンテンツフィルタリングと禁止トピックルールを設定し、有害・不適切な出力をブロックする
  5. E. モデルのパラメータ数が多いほど必ず品質が向上するため、最大パラメータのモデルのみを選択する
解答と解説を見る

正解: C, D

本番環境のリスク管理には、技術的ガードレール（Bedrock Guardrails）と人間評価（Human Evaluation）の組み合わせが業界標準。Bは誤り：temperature を 2.0 以上にすると出力が非常にランダム・不整合になり、品質が低下する。Aは誤り：単一スレッド処理は並行リクエスト時のスループットを著しく低下させ、本番環境のスケーラビリティ要件を満たせない。Eは誤り：大規模モデルが高品質とは限らず（過剰品質・コスト増）、タスクの性質・ユースケースに合ったモデル選定が重要。また最大パラメータのモデルが提供されているとは限らない。
