AIF-C01基盤モデルの応用MEDIUM単一選択

ある EC サイトが Amazon Bedrock のエージェント機能を使って「在庫照会 → 注文確認 → 出荷手配」を自動実行するカスタマーサービスボットを構築しようとしている。エージェントが各ステップで適切な API を呼び出せるようにするために最も重要な設定はどれか。

  1. A. temperature を 0 に設定してエージェントの動作を完全に決定論的にする
  2. B. エージェントの memory 機能を有効化して会話履歴を保持させる
  3. C. Amazon Bedrock Knowledge Bases を設定して API ドキュメントをベクトル検索可能にする
  4. D. OpenAPI スキーマを使ったアクショングループを定義し、呼び出すべき API の仕様をエージェントに登録する
解答と解説を見る

正解: D

Amazon Bedrock エージェントでは「アクショングループ」に OpenAPI スキーマを登録することで、エージェントがどの API をどのパラメータで呼び出せるかを理解し、ユーザーの意図に応じて自律的に API を選択・実行できるようになる。これがエージェントの外部システム連携の核心。Bのメモリ機能は複数ターンの会話文脈を維持するためのもので、API 呼び出し能力の付与とは別の機能。Aの temperature 設定はエージェントのランダム性を制御するが、API 呼び出しの「能力」そのものとは無関係で、正確な API 呼び出しには構造化されたアクショングループ定義が必要。Cの Knowledge Bases は文書検索（RAG）に使うものでありAPI呼び出しの仕組みとは別物。
