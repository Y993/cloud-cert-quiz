AIF-C01AI と ML の基礎MEDIUM単一選択

あるデータサイエンティストが Amazon SageMaker Autopilot を使ってモデルを構築しました。Autopilot はどのような作業を自動化しますか。最も適切な説明を選んでください。

  1. A. 特徴エンジニアリング、アルゴリズム選択、ハイパーパラメータチューニングを自動化し、最良モデルを提案する
  2. B. モデルの解釈可能性レポートのみを生成し、トレーニングは手動で行う必要がある
  3. C. SageMaker Studio のUI操作をスクリプト化して再現性を高める機能である
  4. D. データ収集からラベリング、モデルのデプロイ後の収益化まですべてを自動化する
解答と解説を見る

正解: A

Amazon SageMaker Autopilot は AutoML サービスであり、入力データセットに対して特徴エンジニアリング・アルゴリズム候補の探索・ハイパーパラメータチューニングを自動実行し、最も性能の高いモデルとそのノートブックコードを提供します。データの収集・ラベリング作業はユーザーが準備する必要があり、Autopilot の対象外です。解釈可能性レポートは Autopilot の副産物として提供されますが、主機能はモデルの自動構築です。Studio のUI操作をスクリプト化する機能は SageMaker Pipelines や SDK が担う部分であり、Autopilot の説明としては誤りです。
