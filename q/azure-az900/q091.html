AZ-900Azureの管理とガバナンスEASY単一選択

ある企業がAzureに移行後、毎月のAzure請求書の内訳を確認し、どのサービスがどれだけのコストを占めているかを詳細に分析したい。使用するべきAzureのツールはどれか。

  1. A. Azure Advisor：最適化の推奨事項を確認する
  2. B. Azure 料金計算ツール（Pricing Calculator）：将来の見積もりを作成する
  3. C. Azure Monitor：リソースのメトリクスとログを確認する
  4. D. Azure Cost Management + Billingのコスト分析：リソースグループ・サービス・タグ・期間でフィルタリングして実コストを可視化する
解答と解説を見る

正解: D

Azure Cost Management + Billingのコスト分析（Cost Analysis）機能は、実際の使用コストをサービス別・リソースグループ別・サブスクリプション別・タグ別・リージョン別・期間別などのディメンションでフィルタリング・グルーピングして可視化するツールである。コストの内訳を棒グラフ・折れ線グラフ・テーブルで確認でき、CSVエクスポートも可能。選択肢BのPricing Calculatorは将来の見積もりに使うツールであり、過去の実コスト分析には使用しない。選択肢AAzure AdvisorはAzureリソースの改善推奨を提供するが、月次コストの内訳詳細分析機能はCost Managementが担当する。選択肢CAzure Monitorはパフォーマンスメトリクス・ログ監視のサービスであり、コストの内訳分析には使用しない。
