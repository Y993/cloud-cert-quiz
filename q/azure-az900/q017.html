AZ-900クラウドの概念HARD単一選択

ある企業のクラウドアーキテクトが「クラウドの予測可能性（Predictability）」について経営陣に説明している。Azureにおける「コストの予測可能性」を高めるためのツールや機能の組み合わせとして最も適切なものはどれか。

  1. A. Azure AdvisorとAzure Service Health
  2. B. Azure Pricing CalculatorとAzure Cost Management + Billing
  3. C. Azure MonitorとAzure Security Center
  4. D. Azure BlueprintsとAzure Policy
解答と解説を見る

正解: B

コストの予測可能性を高めるには、事前見積もりと実費管理の両方が必要である。Azure Pricing Calculator（料金計算ツール）は構成するリソースの月次コストを事前に見積もることができ、Azure Cost Management + Billingは実際の支出のモニタリング・予算アラート・コスト分析を提供する。この2つを組み合わせることでコストを計画・管理・予測できる。選択肢CのAzure MonitorはアプリケーションやインフラのパフォーマンスKPI監視ツール、Azure Security Centerはセキュリティ姿勢管理ツールであり、コスト予測には直接使用しない。選択肢DのAzure Blueprintsはガバナンスポリシーのデプロイ自動化、Azure PolicyはAzureリソースへのルール適用であり、コスト予測ではなくガバナンスに関する機能である。選択肢AのAzure Advisorはコスト最適化の推奨を含むが、Cost Managementと組み合わせて使うものであり、料金計算ツールの代替にはならない。
