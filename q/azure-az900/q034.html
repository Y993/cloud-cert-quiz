AZ-900AzureのアーキテクチャとサービスMEDIUM単一選択

ある企業がオンプレミスのMicrosoft SQL Serverデータベースをできる限り変更なくAzureに移行したい。フルマネージドで、OSやSQL Serverのパッチ適用をAzureに任せたい。既存のSQL Server機能との高い互換性も求められる。最も適切なAzureデータサービスはどれか。

  1. A. Azure Virtual Machines上のSQL Server：IaaSとして自己管理するSQL Server
  2. B. Azure SQL Managed Instance：オンプレミスSQL Serverとほぼ完全な互換性を持つフルマネージドSQLサービス
  3. C. Azure SQL Database（シングルデータベース）：フルマネージドのPaaS型SQLデータベース
  4. D. Azure Cosmos DB：グローバル分散型のNoSQLデータベース
解答と解説を見る

正解: B

Azure SQL Managed Instanceは、SQL Server Agentやリンクサーバー・CLR・Service BrokerなどオンプレミスのSQL Serverで使われる高度な機能を含む、ほぼ完全な互換性を持つフルマネージドサービスである。OSとSQL Serverのパッチ適用・バックアップ・高可用性設定をAzureが管理するため、移行後の管理負荷を最小化できる。選択肢DのCosmos DBはNoSQLデータベースであり、SQL Serverからの移行はスキーマ・クエリの大幅な書き換えが必要。選択肢CのAzure SQL Databaseはマネージドだが、Managed Instanceと比べるとSQL Serverの一部の機能（CLRなど）が制限されており、完全互換とは言えない。選択肢AのIaaS上のSQL Serverは互換性は最高だが、OSとSQL Serverのパッチ適用は顧客の責任で管理負荷が高い。
