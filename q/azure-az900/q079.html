AZ-900Azureの管理とガバナンスEASY単一選択

ある企業がAzureのサポートプランを選択しようとしている。本番環境で問題が発生した際に24時間365日・1時間以内の初回応答時間が必要である。この要件を満たすAzureサポートプランはどれか。

  1. A. Basic（基本）：無料で含まれるセルフサービスリソースへのアクセスのみ
  2. B. Developer（開発者）：8〜24時間の応答時間・ビジネス時間帯のみ
  3. C. Standard（標準）：本番環境サポート・重大度Aで1時間以内の応答
  4. D. Premier（プレミア）：Microsoftのエンジニアが専任対応（現在はUnifiedに統合）
解答と解説を見る

正解: C

Azure Standard（標準）サポートプランは本番環境のサポートを提供し、重大度A（クリティカルな本番停止）の場合に1時間以内の初回応答時間（24時間365日）を保証している。本番環境で問題が発生した際の迅速なサポートが必要なビジネスに適している。選択肢AのBasicはポータル・ドキュメント・コミュニティフォーラムのみで、24時間サポートへのアクセスはない。選択肢BのDeveloperはビジネス時間帯のメールサポートのみで、24時間対応ではなく本番環境の重大度Aには対応していない。選択肢DのPremier（現在はUnified Premier/Unifiedサポートに統合）はより高度な専任対応だが、Standardが「1時間以内・24時間365日」という要件を満たす最低限のプランとして正しい答えである。
