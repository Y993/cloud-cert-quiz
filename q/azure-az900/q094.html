AZ-900AzureのアーキテクチャとサービスEASY単一選択

ある企業がAzureの「サブスクリプション」の役割を理解しようとしている。Azureサブスクリプションについての正しい説明はどれか。

  1. A. Microsoft Entra ID（旧 Azure AD）テナントと完全に同一の概念
  2. B. AzureポータルへのログインIDとパスワードの組み合わせ
  3. C. Azureサービスの使用に対する課金および管理の単位であり、リソースをデプロイするための論理的なコンテナ
  4. D. Azureリソースを物理的に分離するためのデータセンターの単位
解答と解説を見る

正解: C

Azureサブスクリプションは課金（利用料金の支払い先）・管理（リソースの論理的なコンテナ）・アクセス制御（RBACのスコープ）の単位として機能する。一つのMicrosoft Entra IDテナントに複数のサブスクリプションを関連付けることができ、サブスクリプションごとに別の支払い方法や部門予算を割り当てられる。選択肢Dの物理的なデータセンター分離はリージョンや可用性ゾーンの概念であり、サブスクリプションは論理的な単位である。選択肢AのEntra IDテナントは組織のIDディレクトリ（ユーザー・グループ管理）であり、サブスクリプションとは別の概念（1テナントに複数サブスクリプションを紐付け可能）。選択肢BはログインIDとパスワードはユーザーアカウントの話であり、サブスクリプションの定義ではない。
