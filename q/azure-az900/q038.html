AZ-900AzureのアーキテクチャとサービスMEDIUM単一選択

ある開発チームがAzureで短時間（最大数分）だけ実行される処理を構築している。サーバーのプロビジョニングや管理は一切行いたくなく、コードが実行された時間分だけ課金されるサーバーレス実行環境が必要である。最も適切なAzureサービスはどれか。

  1. A. Azure Virtual Machines：仮想マシンを自分でプロビジョニングして管理する
  2. B. Azure Functions：イベントトリガーで実行されるサーバーレス関数実行環境
  3. C. Azure App Service（Basic プラン）：常時稼働のWebアプリホスティング環境
  4. D. Azure Kubernetes Service（AKS）：Kubernetesによるコンテナオーケストレーション
解答と解説を見る

正解: B

Azure Functionsはサーバーレスのコンピューティングサービスで、HTTPリクエスト・タイマー・キューメッセージなどのイベントをトリガーとしてコードが実行され、実行時間分だけ課金される（従量課金プランの場合）。サーバーのプロビジョニング・OS管理・パッチ適用はAzureが担当する。選択肢AのVMはサーバーの全管理責任が顧客にあり、停止中も課金が発生する（割り当て解除しない場合）。選択肢CのApp Service BasicプランはWebアプリの常時稼働環境で、使用量に関わらず月額固定料金が発生する。選択肢BのAKSはKubernetesクラスターの管理が必要で、サーバーレスとは言えない。
