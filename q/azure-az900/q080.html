AZ-900Azureの管理とガバナンスMEDIUM単一選択

ある企業がAzureで本番Webアプリケーションを運用しており、特定のAzureリージョンでサービス障害が発生した場合に他のリージョンへ自動フェールオーバーしたい。さらに、世界中のユーザーへの低レイテンシーな配信とWebアプリケーションファイアウォール（WAF）機能も必要である。これらすべての要件を1つのサービスで実現できるものはどれか。

  1. A. Azure Load Balancer：リージョン内のL4負荷分散サービス
  2. B. Azure Traffic Manager：DNS ベースのグローバルトラフィックルーティングサービス
  3. C. Azure Application Gateway：リージョン内のL7ロードバランサーとWAF
  4. D. Azure Front Door：グローバルなHTTP/HTTPSロードバランシング・リージョン間フェールオーバー・CDN・WAF統合サービス
解答と解説を見る

正解: D

Azure Front DoorはMicrosoftのグローバルエッジネットワーク（Anycastネットワーク）を活用したグローバルHTTP/HTTPSロードバランシングサービスで、リージョン間の自動フェールオーバー・世界中のユーザーへの低レイテンシー配信（エッジPOPでのキャッシュ）・Web Application Firewall（WAF）機能をすべて1つのサービスで提供する。選択肢AAzure Load BalancerはL4（TCP/UDP）の同一リージョン内の負荷分散のみで、リージョン間フェールオーバー・グローバル配信・WAFはない。選択肢BAzure Traffic ManagerはDNSベースのグローバルルーティングサービスで、フェールオーバーとグローバルルーティングは可能だが、CDN機能・WAF機能はない。選択肢CAzure Application GatewayはリージョンスコープのL7ロードバランサー・WAFで、グローバル分散やCDN機能はない。
