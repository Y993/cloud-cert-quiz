AZ-900Azureの管理とガバナンスMEDIUM単一選択

ある企業がAzureを利用しており、Azureのサービスに障害が発生した際に影響を受けているサービスと推奨アクションをすぐに確認したい。また、定期メンテナンスの予告も事前に把握したい。最も適切なAzureサービスはどれか。

  1. A. Azure Service Health：Azureサービスの正常性・障害・計画メンテナンス情報
  2. B. Azure Monitor：カスタムメトリクス監視とアラート
  3. C. Microsoft Defender for Cloud：セキュリティ態勢評価
  4. D. Azure Advisor：パフォーマンス・コスト・セキュリティの推奨提供
解答と解説を見る

正解: A

Azure Service HealthはAzureサービス自体の障害（インシデント）・計画メンテナンス（サービス低下・退職予告）・ヘルスアドバイザリーの情報を提供する。影響を受けているAzureリージョンとサービスを絞り込んで確認でき、アラートを設定すれば障害発生・メンテナンス開始時にメールやSMSで通知される。選択肢BAzure Monitorは顧客自身のリソースのパフォーマンスを監視するサービスで、Azure基盤の障害情報を取得するものではない。選択肢DAzure Advisorはコスト・セキュリティ・パフォーマンス・信頼性の改善推奨を定期的に提供するサービスで、リアルタイムの障害情報や計画メンテナンス通知は担当しない。選択肢CのDefender for Cloudはセキュリティ評価・脅威防御のサービスであり、Azureサービス自体の正常性情報とは無関係。
