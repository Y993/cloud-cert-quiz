AZ-900Azureの管理とガバナンスMEDIUM単一選択

ある企業のITチームがAzureリソースのコスト最適化・パフォーマンス改善・セキュリティ強化・信頼性向上のための推奨事項を自動で受け取りたい。Azureが利用パターンを分析して改善提案を行うサービスはどれか。

  1. A. Azure Monitor：メトリクスとログの収集・分析・アラート
  2. B. Azure Policy：リソースのコンプライアンスルールを強制する
  3. C. Azure Advisor：Azureリソースの使用状況を分析してコスト・セキュリティ・信頼性・パフォーマンス・オペレーショナルエクセレンスの推奨を提供
  4. D. Azure Cost Management + Billing：コスト追跡と予算管理
解答と解説を見る

正解: C

Azure Advisorは顧客のAzureリソースの設定と使用状況を継続的に分析し、コスト（未使用リソースの削除・VM右サイジング・予約インスタンス推奨）・セキュリティ（MFA未設定・公開エンドポイントの修正）・信頼性（単一インスタンスVM・バックアップ未設定）・パフォーマンス（データベースのインデックス最適化）・オペレーショナルエクセレンスの5カテゴリで具体的な改善提案を行う。選択肢DAzure Cost Management + Billingはコスト追跡・予算管理・コスト分析の機能を持つが、リソースの使用パターン分析による改善推奨はAdvisorの役割。選択肢BAzure Policyはルールに基づいてリソースへのコンプライアンス強制を行うが、使用パターンを分析した改善推奨は提供しない。選択肢AAzure Monitorはリアルタイムのメトリクス・ログ監視でアラート設定に使うが、利用状況分析による推奨提供機能はAdvisorが担当する。
