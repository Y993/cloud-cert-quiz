AZ-900Azureの管理とガバナンスMEDIUM単一選択

ある企業がAzureで「ゼロダウンタイム」に近い形でAzure SQLデータベースのパフォーマンス問題を調査したい。データベースの現在のDTU（データベーストランザクションユニット）消費率・クエリのパフォーマンス統計・インデックス最適化の推奨をリアルタイムで確認したい。最も適切なAzureの機能はどれか。

  1. A. Azure Advisor：Azureリソース全体の改善推奨を提供
  2. B. Azure SQL Database Query Performance Insights（クエリパフォーマンスの詳細分析）：リソース消費の多いクエリを特定し、チューニング推奨を表示
  3. C. Azure Service Health：Azureサービス自体の正常性情報
  4. D. Azure Cost Management：コストの追跡と予算管理
解答と解説を見る

正解: B

Azure SQL DatabaseのQuery Performance Insights（QPI）はAzureポータルからアクセスできるデータベース固有のパフォーマンス分析ツールで、CPU・データ入出力・ログの消費量が多いクエリをランキング表示し、特定のクエリの実行計画やパフォーマンス履歴を可視化できる。また、Automatic Tuning機能と組み合わせてインデックス最適化の推奨事項も提示する。選択肢AAzure Advisorはコスト・セキュリティ・信頼性・パフォーマンスの一般推奨を提供するが、SQLクエリレベルの詳細パフォーマンス分析（どのクエリが重いか）はQuery Performance Insightsの専門機能。選択肢DAzure Cost Managementはコスト追跡ツールであり、データベースのクエリパフォーマンス分析とは無関係。選択肢CAzure Service HealthはAzure基盤の正常性情報であり、特定データベースのクエリパフォーマンス分析には使用しない。
