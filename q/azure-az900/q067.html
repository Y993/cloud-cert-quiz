AZ-900Azureの管理とガバナンスMEDIUM単一選択

ある企業がWindowsサーバーライセンスを大量に保有しており、Azureに移行する際にそのライセンスをAzure VMで再利用することでコストを削減したい。このライセンスの再利用を可能にするAzureの特典プログラムはどれか。

  1. A. Azure Free Trial：無料試用期間中のクレジット
  2. B. Azure Dev/Test料金：開発・テスト向けの割引料金
  3. C. Azure スポット料金：余剰キャパシティの低価格利用
  4. D. Azure Hybrid Benefit（Azureハイブリッド特典）：ソフトウェアアシュアランス付きの既存Windowsライセンスをクラウドで再利用できる
解答と解説を見る

正解: D

Azure Hybrid Benefit（Azureハイブリッド特典）は、ソフトウェアアシュアランス（SA）付きのオンプレミスWindowsサーバーまたはSQL Serverライセンスを、Azure VM・SQL Database・SQL Managed Instanceなどのクラウドサービスに適用することで、クラウド側のOSライセンス費用を削減できるプログラムである。Windows ServerとSQL Server両方に適用可能で、コスト削減効果は最大40%以上になる場合もある。選択肢AのAzure Free Trialは新規アカウント向けの初回無料クレジットであり、既存ライセンスの再利用とは無関係。選択肢BAzure Dev/Test料金はVisual Studioサブスクリプション所有者向けの開発・テスト環境の割引で、本番環境での既存ライセンス活用とは異なる。選択肢CのSpotは余剰キャパシティの低価格利用であり、ライセンス活用プログラムではない。
