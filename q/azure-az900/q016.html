AZ-900クラウドの概念MEDIUM単一選択

ある大企業が自社のデータセンターにAzure Stack Hubを導入することを検討している。Azure Stack Hubを採用する主な理由として最も適切なものはどれか。

  1. A. グローバルなAzureリージョンへのアクセスを高速化したい
  2. B. インターネット接続が制限された環境や規制上の理由でオンプレミスにAzureサービスを展開したい
  3. C. パブリッククラウドよりも低コストでAzureサービスを利用したい
  4. D. 複数のクラウドプロバイダーを統合管理したい
解答と解説を見る

正解: B

Azure Stack Hubは、インターネット接続が制限・禁止されている隔離環境（エアギャップ環境）や、データの国外持ち出し規制、低レイテンシーが必要なエッジ環境など、パブリッククラウドへの接続が困難・不可能な場合にオンプレミスでAzureサービスを提供する製品である。選択肢CはAzure Stack Hubがパブリッククラウドより低コストとは限らず、追加のハードウェア投資が必要なため誤り。選択肢AのグローバルAzureリージョンへのアクセス高速化はExpressRouteやAzure FrontDoorの役割であり、Azure Stack Hubとは異なる。選択肢DのマルチクラウドはAzure Arcの機能に近く、Azure Stack HubはAzureサービスのオンプレミス展開に特化している。
