AZ-900AzureのアーキテクチャとサービスEASY単一選択

ある企業がAzureでWebサイトをホストするためにストレージを検討している。HTMLファイル・CSSファイル・画像ファイルなどの静的コンテンツを大量に保存し、低コストで世界中のユーザーに配信したい。最も適切なAzureサービスはどれか。

  1. A. Azure Disk Storage：VMに接続する仮想ディスク
  2. B. Azure SQL Database：リレーショナルデータベースサービス
  3. C. Azure Blob Storage：非構造化データ（画像・動画・ドキュメントなど）を格納するオブジェクトストレージ
  4. D. Azure Files：SMB/NFSプロトコルで共有アクセスできるクラウドファイル共有
解答と解説を見る

正解: C

Azure Blob Storage（Azure ストレージのBlob）はHTMLファイル・画像・動画・ドキュメントなどの非構造化データを保存するオブジェクトストレージサービスである。静的Webサイトのホスティング機能も組み込まれており、Azure CDNと組み合わせることで低コストかつグローバルに配信できる。選択肢BのAzure SQL Databaseはリレーショナルデータベースであり、ファイルの直接ホスティングには使用しない。選択肢DのAzure FilesはSMB/NFS共有ファイルシステムで、ファイルサーバーの置き換えやVMからのマウントに使う。静的コンテンツのWeb配信には最適ではない。選択肢AのAzure Disk Storageは仮想マシンのOSディスク・データディスクとして使用するブロックストレージであり、Webコンテンツの配信には使用しない。
