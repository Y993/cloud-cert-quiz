DP-900Azure の分析ワークロードMEDIUM複数選択

ある企業が Microsoft Fabric を採用してデータ分析基盤を刷新しようとしている。Microsoft Fabric の特徴として正しいものを2つ選択してください。

  1. A. Fabric のレイクハウスは Delta Lake 形式をネイティブにサポートし、SQL と Spark の両方からアクセスできる
  2. B. Fabric はデータエンジニアリング・データサイエンス・データウェアハウス・BI を単一の SaaS プラットフォームに統合している
  3. C. Fabric のセマンティックモデルは Power BI の機能とは独立しており、相互運用性はない
  4. D. Fabric の OneLake は各ワークスペースが独立したストレージを持ち、データを共有するには明示的なコピーが必要である
  5. E. Fabric は Azure Synapse Analytics と完全に同一のサービスであり、機能に違いはない
解答と解説を見る

正解: A, B

Microsoft Fabric は Data Engineering・Data Science・Data Warehouse・Real-Time Analytics・Power BI・Data Factory を単一の SaaS プラットフォームに統合したサービスである（選択肢 B が正解）。また Fabric のレイクハウスは Delta Lake 形式をネイティブサポートし、同一データに SQL エンドポイントと Spark ノートブックの両方からアクセスできる統一アーキテクチャを持つ（選択肢 A が正解）。選択肢 D は誤りで、OneLake は組織全体で1つの論理ストレージを共有するアーキテクチャであり、コピーなしでデータを参照できる。選択肢 E は誤りで、Fabric は Synapse Analytics の後継・進化形として位置づけられており、アーキテクチャや機能セットに明確な違いがある。選択肢 C は誤りで、Fabric のセマンティックモデルは Power BI と深く統合されており、Power BI レポートのデータ基盤として機能する。
