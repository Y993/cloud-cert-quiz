DP-900Azure の分析ワークロードEASY単一選択

Azure のデータレイクとして機能し、階層型名前空間による高性能なファイルシステムアクセスと、大規模な分析ワークロードに最適化されたストレージサービスはどれか。

  1. A. Azure Disk Storage：仮想マシン向けのマネージドディスク
  2. B. Azure Blob Storage（階層型名前空間なし）：汎用オブジェクトストレージ
  3. C. Azure Files：SMB/NFS プロトコル対応のマネージドファイル共有
  4. D. Azure Data Lake Storage Gen2：階層型名前空間対応の分析特化ストレージ
解答と解説を見る

正解: D

Azure Data Lake Storage Gen2（ADLS Gen2）は Azure Blob Storage 上に階層型名前空間（Hierarchical Namespace）を有効化したサービスであり、ディレクトリベースの操作が可能でファイルシステムと同等のパフォーマンスを提供する。Synapse Analytics や Databricks などの分析エンジンと緊密に統合されており、大規模な分析ワークロードに最適である。選択肢 B の階層型名前空間なしの Blob Storage はオブジェクトストレージとして動作するが、ディレクトリ操作のパフォーマンスは ADLS Gen2 より低く、大規模分析には向かない。選択肢 C の Azure Files はファイル共有プロトコル向けであり、分析データレイクの用途には使わない。選択肢 A の Azure Disk Storage は VM 接続ディスク向けであり、データレイクとしては使用しない。
