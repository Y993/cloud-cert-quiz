DP-900Azure の分析ワークロードMEDIUM単一選択

ある企業のアナリストが、Power BI Desktop でレポートを作成してから、同僚がどこからでも閲覧・共有できるように公開したい。また、データは毎朝8時に自動更新される必要がある。この要件を満たすために必要な手順として正しいものはどれか。

  1. A. Power BI Desktop でレポートを作成し、PBIX ファイルをメールで配布する。受信者がそれぞれ手動でデータを更新する
  2. B. Power BI Desktop でレポートを Azure Blob Storage にエクスポートし、URL を共有することでアクセスを提供する
  3. C. Power BI Desktop でレポートを SharePoint に直接アップロードし、SharePoint の自動更新機能を使用する
  4. D. Power BI Desktop でレポートを作成し、Power BI Service に発行（Publish）する。Service 上でスケジュール更新を設定し、アプリまたはワークスペースで共有する
解答と解説を見る

正解: D

Power BI のレポート共有と自動更新は Power BI Service（クラウド版）で提供される機能である。Power BI Desktop でレポートを作成した後、Service に発行（Publish）することでクラウドにアップロードされ、同僚がブラウザやモバイルアプリからアクセスできる。スケジュール更新はService 上のデータセット設定で特定時刻・頻度を指定して自動実行できる。選択肢 A の PBIX ファイルのメール配布では自動更新も集中管理もできず、受信者がそれぞれ Power BI Desktop を持つ必要がある。選択肢 C の SharePoint への直接アップロードは Power BI Service の正規ワークフローではなく、スケジュール更新機能もない。選択肢 B の Blob Storage へのエクスポートはレポートの共有方法ではない。
