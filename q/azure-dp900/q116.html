DP-900Azure の分析ワークロードMEDIUM単一選択

Power BI において、経営幹部向けに KPI 指標（売上・利益・顧客数）を一画面で常時監視できるビューを作成したい。複数のレポートから重要なビジュアルを選んで1か所に集約し、自動更新されるビューを提供するために使用すべき Power BI オブジェクトはどれか。

  1. A. Power BI データフロー：Power Query ベースのクラウド ETL 機能
  2. B. Power BI レポート：複数ページにわたる詳細なデータ可視化ファイル
  3. C. Power BI セマンティックモデル：データモデルとメジャーを定義した再利用可能な基盤
  4. D. Power BI ダッシュボード：複数レポートのビジュアルをタイルとしてピン留めした概要画面
解答と解説を見る

正解: D

Power BI ダッシュボードは Power BI Service 上でのみ作成でき、複数のレポートのビジュアル（グラフ・カード・マップなど）をタイルとしてピン留めして1画面に集約できる。ダッシュボードは常に最新データを表示し、経営幹部が重要 KPI を一覧するモニタリング画面として最適である。選択肢 B のレポートは詳細分析向けの複数ページファイルであり、経営幹部向けの概要モニタリングよりも詳細分析に向く。また複数レポートのビジュアルを1か所に集約する機能は持たない。選択肢 C のセマンティックモデルはデータの論理モデルであり、ビジュアル表示を行うオブジェクトではない。選択肢 A のデータフローはデータ準備の ETL 機能であり、表示画面ではない。
