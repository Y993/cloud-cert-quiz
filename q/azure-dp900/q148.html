DP-900Azure の分析ワークロードHARD複数選択

ある企業が Azure Databricks と Microsoft Fabric のどちらを採用するか検討している。それぞれの特徴について正しいものを2つ選択してください。

  1. A. Azure Databricks は Apache Spark をベースとした高度な機械学習・大規模データ処理に強みがあり、MLflow による実験管理やカスタム Spark 設定の柔軟性が高い
  2. B. Microsoft Fabric は Power BI・Data Factory・Synapse Analytics などを統合した SaaS プラットフォームであり、データエンジニアリングから BI レポートまでを単一環境で提供する
  3. C. Microsoft Fabric は機械学習モデルのトレーニングに特化しており、SQL によるデータ分析はサポートしていない
  4. D. Azure Databricks は Microsoft Fabric の内部コンポーネントであり、独立したサービスとして使用することはできない
  5. E. Azure Databricks のノートブックは SQL のみをサポートし、Python や R は使用できない
解答と解説を見る

正解: A, B

Azure Databricks は Apache Spark をベースとした高度なビッグデータ・機械学習プラットフォームであり、MLflow 統合による ML ライフサイクル管理、Delta Lake、カスタム Spark 設定の柔軟性が特徴である（選択肢 A が正解）。Microsoft Fabric は Power BI・Azure Data Factory・Synapse Analytics・Databricks ランタイムなどを統合した SaaS プラットフォームであり、単一の UI からデータエンジニアリング・データサイエンス・データウェアハウス・BI を管理できる（選択肢 B が正解）。選択肢 D は誤りで、Azure Databricks は Fabric とは独立したサービスとして利用可能である（Fabric に Databricks ランタイムは統合されているが、Databricks 自体は独立して存在する）。選択肢 C は誤りで、Fabric は Lakehouse の SQL エンドポイント・Warehouse・Power BI など、SQL 分析を中心的なワークロードとしてサポートしている。選択肢 E は誤りで、Databricks は Python・R・Scala・SQL の多言語ノートブックをサポートしている。
