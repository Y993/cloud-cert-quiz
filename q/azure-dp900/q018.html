DP-900コア データ概念MEDIUM複数選択

ある企業がデータ活用プロジェクトを進めるにあたり、「データエンジニア」と「データアナリスト」の役割を明確に分担しようとしている。「データアナリスト」が主に担う責務として正しいものを2つ選択してください。

  1. A. Power BI などのBIツールでダッシュボードを作成し、売上・コスト・KPIをビジュアル化して経営陣に提示する
  2. B. SQLを使ってデータを集計・加工し、部門の業務上の問いに答えるレポートを作成する
  3. C. データウェアハウスのパーティション設計やインデックス最適化を行いクエリ性能を向上させる
  4. D. 機械学習モデルをトレーニングして本番環境にデプロイするMLOpsパイプラインを構築する
  5. E. Azure Data Factory を使って複数ソースからデータを取り込むETLパイプラインを設計する
解答と解説を見る

正解: A, B

データアナリストの主な責務は（1）BIツール（Power BIなど）でダッシュボードを作成してビジネス指標を可視化すること、および（2）SQLを使ってデータを集計・加工しビジネス上の問いに答えるレポートを作成することである。どちらも「既存データを分析・可視化してビジネス判断を支援する」という役割を体現している。選択肢EのETLパイプライン設計はデータエンジニアの主な責務である。データアナリストはパイプラインが整備済みのデータを消費する立場となる。選択肢Cのパーティション設計・インデックス最適化はデータベース管理者（DBA）またはデータエンジニアの責務であり、データアナリストが実施する作業ではない。選択肢DのMLOpsパイプライン構築は機械学習エンジニアまたはデータサイエンティストの専門領域である。
