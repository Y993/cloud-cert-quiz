DP-900Azure のリレーショナル データMEDIUM複数選択

ある企業が Azure でリレーショナルデータベースサービスを選定しようとしている。Azure SQL Database と Azure SQL Managed Instance の両方に当てはまる特性を2つ選択してください。

  1. A. 自動バックアップ機能が組み込まれており、ポイントインタイムリストアが可能である
  2. B. クロスデータベースクエリ（同一インスタンス内の別データベースへの参照）を完全サポートする
  3. C. SQL Server Agent ジョブをネイティブにサポートする
  4. D. PaaS（Platform as a Service）形式で提供され、OS の管理が不要である
  5. E. OS レベルのカスタムソフトウェアインストールが可能である
解答と解説を見る

正解: A, D

Azure SQL Database と Azure SQL Managed Instance はどちらも PaaS サービスであり、OS の管理（パッチ適用・セキュリティ更新）は Azure が担うため選択肢Dは両方に該当する。また、両サービスとも自動バックアップとポイントインタイムリストア（PITR）が組み込まれており選択肢Aも両方に該当する。SQL Server Agent のネイティブサポートは Azure SQL Managed Instance のみの特性であり、Azure SQL Database ではサポートされていない（選択肢Cは不正解）。OS レベルのカスタムインストールは IaaS（Azure VM 上の SQL Server）の特性であり、どちらの PaaS サービスでも不可能（選択肢Eは不正解）。クロスデータベースクエリの完全サポートは Managed Instance の特性であり、Azure SQL Database では制限がある（選択肢Bは不正解）。
