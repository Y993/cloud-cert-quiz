DP-900Azure の分析ワークロードEASY単一選択

Power BI において、複数のデータソースを接続し、テーブル間のリレーションシップとメジャーを定義した論理データモデルの名称はどれか。

  1. A. データフロー：Power Query ベースのクラウド側 ETL 処理
  2. B. レポート：グラフや表を配置してデータを可視化したページ集合
  3. C. セマンティックモデル（データセット）：データソース接続・リレーション・メジャーを定義した再利用可能なモデル
  4. D. ダッシュボード：複数のレポートからタイルをピン留めした一覧画面
解答と解説を見る

正解: C

Power BI のセマンティックモデル（旧称：データセット）はデータソースへの接続、テーブル間のリレーションシップ定義、DAX メジャーの定義をまとめた論理データモデルであり、複数のレポートやダッシュボードから再利用できる。選択肢 B のレポートはセマンティックモデルをもとにビジュアルを配置したページ集合であり、モデル自体ではない。選択肢 D のダッシュボードは Power BI Service 上でレポートのビジュアルをタイルとしてピン留めした概要画面であり、レポートとは別のオブジェクトである。選択肢 A のデータフローは Power Query を使ったクラウド側の ETL 機能であり、セマンティックモデルの入力データを準備する役割を持つが、モデル自体ではない。
