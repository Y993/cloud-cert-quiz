DP-900Azure の分析ワークロードMEDIUM単一選択

ある企業が、Azure Data Factory のコピーアクティビティを使って、オンプレミスの SQL Server データベースから Azure Synapse Analytics にデータを移行している。オンプレミスネットワーク内のデータにアクセスするために必要なコンポーネントはどれか。

  1. A. セルフホスト型 Integration Runtime（Self-hosted IR）：オンプレミス環境内にインストールしてデータ移動を実行するエージェント
  2. B. Azure VPN Gateway：サイト間 VPN でネットワークを接続するゲートウェイサービス
  3. C. Azure ExpressRoute：専用線でオンプレミスと Azure を接続するネットワークサービス
  4. D. Azure Active Directory：認証と認可を管理するアイデンティティサービス
解答と解説を見る

正解: A

Azure Data Factory のセルフホスト型 Integration Runtime（Self-hosted IR）はオンプレミスのサーバーやプライベートネットワーク内にインストールするエージェントソフトウェアであり、ファイアウォールの内側にあるオンプレミスのデータソース（SQL Server・Oracle・ファイルシステムなど）への接続を可能にする。ADF が HTTPS を通じてセルフホスト IR に指示を送り、IR がオンプレミスデータを取得して Azure に転送する。選択肢 C の ExpressRoute は専用線ネットワーク接続であり、ADF がオンプレミスデータにアクセスするための ADF 固有のエージェント機能ではない。選択肢 B の VPN Gateway もネットワーク接続サービスであり、ADF がオンプレミスに直接接続するためのコンポーネントではない。選択肢 D の Azure Active Directory は認証基盤であり、データ移動のエージェントではない。
