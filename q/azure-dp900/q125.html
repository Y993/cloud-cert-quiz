DP-900Azure の分析ワークロードMEDIUM複数選択

ある企業が Power BI のセマンティックモデルを活用したレポート開発の標準化を進めている。セマンティックモデルの特徴と活用方法として正しいものを2つ選択してください。

  1. A. セマンティックモデルはリレーショナルデータベースのスキーマと完全に同一であり、変換処理を含められない
  2. B. セマンティックモデルを共有データ基盤として運用することで、レポート間でのメトリクス定義の一貫性を保てる
  3. C. セマンティックモデルに定義された DAX メジャーは、そのモデルを参照するすべてのレポートで再利用できる
  4. D. セマンティックモデルは1つの Power BI レポートにのみ紐づけ可能であり、複数レポートからの共有はできない
  5. E. セマンティックモデルは Power BI Desktop 専用の機能であり、Power BI Service では使用できない
解答と解説を見る

正解: B, C

セマンティックモデルに定義された DAX メジャー（売上合計・利益率など）は、そのモデルを参照するすべての Power BI レポートで再利用できる（選択肢 C が正解）。また、共有セマンティックモデルを組織の単一データ基盤として運用することで、複数チームが作成するレポート間でメトリクスの定義・計算ロジックを統一し一貫性を保てる（選択肢 B が正解）。選択肢 D は誤りで、セマンティックモデルは複数のレポートから参照・再利用できることが主要な利点の一つである。選択肢 E は誤りで、セマンティックモデルは Power BI Service 上でも作成・管理・共有できる。選択肢 A は誤りで、セマンティックモデルは Power Query による変換・クリーニング処理やカスタム計算列・DAX メジャーを含めることができる。
